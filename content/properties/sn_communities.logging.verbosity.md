description: "Code Search log verbosity. Set to 'error' by default. Options are: off : Do not generate any logs. error : Show only critical errors which may prevent the search from completing will be shown. warn : Show warning which indicates possible unexpected changes in behavior while searching, in addition to any errors. info : Show progress messages as the Code Search application searches for results. Also shows warnings and error messages. debug : Show information that may be useful while debugging the application, like the values of queries run. Also includes info, warn, and error messages."
