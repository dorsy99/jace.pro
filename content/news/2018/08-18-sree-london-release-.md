<p><strong>Discovery Core Engine updates</strong></p>
<p>Windows PowerShell enthusiasts are in for a real surprise. London release supports PowerShell Remoting protocol and Microsoft JEA (Just Enough Administration). Discovering nodes with windows operating system is going to be twice faster and secure than conventional methods.</p>
<p>By default, to connect to a remote computer using PowerShell (PowerShell Remoting) you need the administrator privileges. Customers can add a simple configuration to allow remote connection using PowerShell Remoting (WinRM) for common users (without the administrator privileges) with group policy settings.</p>
<p>With the London release, Cloud Discovery supports AWS Organizations through a master account discovery. It helps in the easy population of member accounts and discovery of member account resources using dynamically acquired credentials.  </p>
<p> Horizontal discovery provides a brand-new HTTP classification which can find new devices via HTTP protocol.</p>
<h2>What’s new with Service Mapping</h2>
<p><em>“Sometimes you can&#39;t see something for what it is because you are in the midst of it. Maybe you need to get away to gain the right perspective” </em></p>
<p>This is absolutely true in the IT Ops world. Getting visibility to business applications and its dependency to the code IT infrastructure is the tough nut to crack. Service Mapping continuously monitors the hybrid IT infrastructure for service affecting changes and updates service maps in real time with top down service modeling approach. With Kingston/London release, Service Maps are getting simpler and smarter with machine learning. The ML based machine learning algorithms will perform noise reduction especially when you want to visualize TCP IP connections in the maps. Bulk mapping experience now include bulk error handling which simplifies map administration.</p>

<h2><span style="background: white;">What’s new with CMDB</span></h2>
<p style="margin: 0in; font-family: &#39;Helvetica Neue&#39;; font-size: 10.0pt; color: #212121;"> </p>
<p style="margin: 0in; font-family: &#39;Helvetica Neue&#39;; font-size: 10.0pt; color: #212121;"><span style="font-weight: bold; background: white;">Duplicate Resolver</span></p>
<p style="margin: 0in; font-family: &#39;Helvetica Neue&#39;; font-size: 10.0pt; color: #212121;"><span style="background: white;"> </span></p>
<p style="margin: 0in; font-family: &#39;Helvetica Neue&#39;; font-size: 10.0pt; color: #212121;"><span style="background: white;">CMDB acts as a Central Integration Agency for several organizations. CI Duplication is a mammoth challenge that arise where customers struggle to get visibility to CI duplicates and remediate duplicate CIs. CMDB dashboards display CMDB health reports and let you configure the CMDB health KPIs and metrics that configuration Items (CIs) are evaluated for. Health dashboards provide visibility to duplicate CIs. With London release, customers will be able to remediate CMDB CI duplicates with a prescriptive out of box workflows, which helps in merger of duplicate CI and its attributes. </span></p>
<p style="margin: 0in; font-family: &#39;Helvetica Neue&#39;; font-size: 10.0pt; color: #212121;"> </p>
<p style="margin: 0in; font-family: &#39;Helvetica Neue&#39;; font-size: 13.0pt; color: #333333;"><img src="dcf0aaecdbc82b80a39a0b55ca9619aa.iix" width="653" height="206" /></p>

<p style="margin: 0in; font-family: &#39;Helvetica Neue&#39;; font-size: 13.0pt; color: #333333;"><img src="d501a220db0c2b80a39a0b55ca961981.iix" width="650" height="248" /></p>
<p style="margin: 0in; font-family: &#39;Helvetica Neue&#39;; font-size: 13.0pt; color: #333333;"> </p>
<p style="margin: 0in; font-family: &#39;Helvetica Neue&#39;; font-size: 13.0pt; color: #333333;"><span style="font-weight: bold; background: white;">Better together with Software Asset Management / Change Management / IT Business Management</span></p>
<p style="margin: 0in; font-family: &#39;Helvetica Neue&#39;; font-size: 13.0pt; color: #333333;"> </p>
<p>ServiceNow is mastering the “Value as a Service” strategy with its platform centric approach and “Better Together” play. CMDB and Discovery is very well known for its key value proposition of discovering the core infrastructure and application dependencies. The value of discovered data is unlocked from the apps like software asset management, IT operations management, IT service management and IT business management </p>
<p>With Kingston release, we released CMDB SDK and Discovery APIs. With London release, discovery is embedded in to change management process, which makes the CMDB up-to date with the real time discovered CIs. Customers can trigger an on-remand discovery on the affected configuration Items from the change management screens or can auto trigger the discovery based on the state of the change request.</p>
<p><img src="7a316a20db0c2b80a39a0b55ca961992.iix" width="739" height="353" /></p>
<p style="text-align: center;">Image : Change property and real time discovery</p>
<p style="text-align: center;"> </p>
<p style="margin: 0in; font-family: &#39;Helvetica Neue&#39;; font-size: 10.0pt; color: #212121;"><span style="background: white;">With London release, software asset management product released a publisher pack for Citrix. Tracking citrix licensing is challenging, however many customers see compliance for licensable products deployed in a citrix farm far more challenging than citrix license compliance. For London release, using ServiceNow discovery product, you can detect citrix license server and collect concurrent license usage for SAM professional plugin. Also, you will be able to detect Citrix XenApp instance and deployed applications, unique users and sessions for SAM.</span></p>
