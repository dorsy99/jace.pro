<h2>1. Stunning new catalog experience on Service Portal</h2>
<p>In the new release, service catalog experience on Portal has got a complete overhaul.</p>
<ul><li>Modern UI</li><li>New Widgets</li><li>&#34;Show More&#34; option</li></ul>
<p>     <img style="max-width: 100%; max-height: 480px; display: block; margin-left: auto; margin-right: auto;" src="ba1e41dbdb1097444e1df4621f96199e.iix" /></p>

<h3>Service Catalog&#39;s modern UI in Kingston</h3>
<p>One of the themes of this release was to modernize the user experience.</p>
<p><img style="max-width: 100%; max-height: 480px; display: block; margin-left: auto; margin-right: auto;" src="ac7e455fdb1097444e1df4621f9619e2.iix" /> </p>
<p>You will see a modern landing page focused on search-first experience. However, browsing by categories is still available. New category widget provides a more performant as well as modern experience that makes it intuitive for users to browse through the categories.</p>
<p><img class="image-3 jive-image" style="display: block; margin-left: auto; margin-right: auto; width: 386px; height: 148.936px;" src="6d579946db90d7041dcaf3231f961927.iix" alt="browseByCategories.jpg" width="386" height="149" /></p>
<p> </p>
<p>To support the search-first experience, there have been improvements to Search performance that I will talk about in a bit.</p>
<p> </p>
<p>On the landing page, you will find many new widgets to surface status of requests submitted by the users, easy access to approvals assigned to the users and access to recent items specific to the user as well as most popular items across the organization. For example, <strong>My Requests, Saved Bundles, Popular Items</strong> or <strong>My Recent Items</strong>. All of these widgets are ready for mobile web which will give users a consistent and better-looking experience whether they are on desktop or mobile devices.</p>
<p><img class="image-4 jive-image" style="display: block; margin-left: auto; margin-right: auto; width: 470px; height: 329.758px;" src="f456340adb9097041dcaf3231f96199e.iix" alt="widgets.jpg" width="470" height="330" /></p>

<p>Every widget has got a number of instance options to tailor the widgets according to your needs.</p>
<p style="text-align: center;"><img class="image-6 jive-image" style="width: 515px; height: 164.468px;" src="9eeb7ff9db545fc03eb27a9e0f961998.iix" alt="InstanceOptions.jpg" width="515" height="164" /></p>

<h2>2. Order Guides on Service Portal</h2>
<p><a title="ocs.servicenow.com/bundle/kingston-it-service-management/page/product/service-catalog-management/concept/c_ServiceCatalogOrderGuides.html" href="https://docs.servicenow.com/bundle/kingston-it-service-management/page/product/service-catalog-management/concept/c_ServiceCatalogOrderGuides.html" rel="nofollow">Order guide&#39;s</a> familiar 3-stage guided navigation feature will be introduced on Service Portal with Kingston release.</p>
<p><img class="image-8 jive-image" style="display: block; margin-left: auto; margin-right: auto; width: 507px; height: 110.395px;" src="ff0f594adbdc97049c9ffb651f96198c.iix" alt="orderGuide3stage.jpg" width="507" height="110" />In the new order guide widget, you can remove the item or change variable values right on the choose options screen rather than going back to item page.</p>
<p style="text-align: center;"><img class="image-9 jive-image" style="height: 201px; width: 526.946px;" src="553b388edb101b04ed6af3231f961909.iix" alt="ChooseOptions.jpg" width="526" height="201" /></p>
<p> </p>
<p>Clicking the options button will allow you to view and edit the variables values that were entered. Switching the toggle will either include or remove the item from the order. You can allow this toggle button to appear on specific order guides and hide from others. This is a setting when you are defining the order guide:</p>
<p><img class="image-20 jive-image" style="display: block; margin-left: auto; margin-right: auto; width: 574px; height: 188.865px;" src="3446ed8adb1053043eb27a9e0f9619e9.iix" alt="toggle.png" width="574" height="189" /></p>
<p>This widget will also support attachments, which has been a major ask from our users. You will be able to upload attachments to individual items in the order guide which is like icing on the cake or cherry in my bourbon.</p>
<p><img class="image-10 jive-image" style="display: block; margin-left: auto; margin-right: auto; width: 574px; height: 208.306px;" src="214ad886db5c17049c9ffb651f96195c.iix" alt="attchment.jpg" width="574" height="208" /></p>
<p> </p>
<p>Just before submitting the order guide, there is a Summary page which gives the requestor an overview of the entire order and the ability to review the items included before submitting the order or go back to edit options if needed. &#34;Add to cart&#34; is now available on Order guides as well.</p>
<p> </p>
<p><img class="image-11 jive-image" style="display: block; margin-left: auto; margin-right: auto; width: 515px; height: 274.113px;" src="2c510d02dbd097049c9ffb651f96193d.iix" alt="editOptinsOrderGuide.jpg" width="515" height="274" /></p>

<h2>3. Enhanced search experience</h2>
<p>To elevate the search experience on portal, we made changes to the catalog search capability. The new enhanced search should present more relevant results and much quicker than ever before. Search will execute on very specific fields within the service catalog tables and each field is also weighted for its relevance.</p>
<p> </p>
<p>Below are the fields in focus for search:</p>
<ul style="list-style-type: disc;"><li>Name of the catalog item</li><li>Meta field, which holds the search tags</li><li>Short description</li><li>Description</li><li>Category and catalog where the item resides.</li></ul>
<p> </p>
<p>And... we have lightened the cache which should produce results faster than before.</p>
<p> </p>
<h2>4. Performance improvements</h2>
<p>There are significantly noticeable improvements in performance in Kingston. I will avoid going into technical depths; however, to give you an idea:</p>
<p> </p>
<ul><li>Catalog cache memory utilization is reduced</li><li>Client script execution has become faster</li><li>And we have reduced Angular watchers to improve performance on Service Portal.</li></ul>

<p>Here are 5 other enhancements that, though, may be small, definitely enhance the experience.</p>
<p> </p>
<p style="text-align: justify;"><strong>1.
