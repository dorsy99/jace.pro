<p>The DevOps product includes some major new capabilities for ServiceNow® DevOps and will be delivered through the ServiceNow Store close to the Orlando platform release. ServiceNow DevOps has new integrations to Azure DevOps (ADO) in three areas: ADO Boards for planning, ADO Repos for code check-ins, and ADO Pipelines for the build process. Now you can see insights into Azure DevOps pipelines alongside Jenkins-based pipelines and bring ADO information into the automated change creation and decision-making process. </p>
<p><span class="TextRun  BCX0 SCXW139175754" lang="EN-US"><span class="NormalTextRun  BCX0 SCXW139175754">We’re also adding a new reporting dashboard that covers industry standard KPIs as defined in the Accelerate book and promoted by the Google DevOps Research and Assessment organization.</span></span></p>
<p><span class="TextRun  BCX0 SCXW139175754" lang="EN-US"><span class="NormalTextRun  BCX0 SCXW139175754"><img src="https://community.servicenow.com/071ef6fedb8bc814d82ffb243996192d.iix" width="278" height="151" /></span></span></p>
<p>At the same time, we’re extending the pipeline management to include full releases by leveraging artifacts. Now, as you bring together multiple changes into releases from multiple pipelines, we can gain new insights for the management and automation of the end-to-end process. </p>
<p>A new integrations API was also added which makes it easier than ever for customers and ServiceNow partners to add your own DevOps tools to the process.  </p>
<p>This version will be supported on Orlando and New York Now Platform® release levels. </p>
<p>Check out the <a href="https://community.servicenow.com/community?id&#61;community_blog&amp;sys_id&#61;6089aaf0dbba0898d58ea345ca961970" target="_blank" rel="noopener noreferrer nofollow">Now Platform blog</a> for more news about a new integration with CI/CD. That integration makes it easier to apply DevOps processes to developing code on the Now Platform. That means your Now Platform development can be managed in tools supported by the ServiceNow DevOps product like Git repositories and Jenkins enabling the same automated changes, insights, and audit controls. </p>
<p>The other big news is that the DevOps solution is now available in the FedRAMP approved Orlando release, in addition to the version released through the ServiceNow Store*. This means the necessary certification to be compliant with the US Federal Risk and Authorization Management Program (<a href="https://www.fedramp.gov/" rel="nofollow">FedRAMP</a>) is in place for the DevOps solution in the Orlando release – a rare achievement among DevOps management solutions.   </p>
<p><img src="https://community.servicenow.com/300f7272dbcbc814d82ffb243996192f.iix" /></p>
<p>* FedRAMP-compliant product version is v1.8 and has a different set of integrations and capabilities than the most-current store release. </p>
