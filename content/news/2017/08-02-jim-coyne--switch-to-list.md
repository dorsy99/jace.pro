<h5>Included in the <a href="community?id&#61;community_blog&amp;sys_id&#61;b3c843aadb892b40fece0b55ca961906" rel="nofollow">Developer Toolbox</a> Update Set available on Share (link in the &#34;Developer Toolbox&#34; post).</h5>
<p> </p>
<p style="font-family: arial, sans-serif; color: #666666;">Here is &#34;Switch to List View&#34;.   It&#39;s a &#34;Form context menu&#34; UI Action that simply switches to the list view for the table, and filters the list to only display the current record:</p>
<p style="font-family: arial, sans-serif; color: #666666;"> </p>
<p style="font-family: arial, sans-serif; color: #666666;"><img class="image-3 jive-image" style="max-width: 1200px; max-height: 900px;" src="0a69c882db5cdb048c8ef4621f961916.iix" /></p>
<p style="font-family: arial, sans-serif; color: #666666;"> </p>
<p style="font-family: arial, sans-serif; color: #666666;"><img class="image-2 jive-image" style="max-width: 1200px; max-height: 900px;" src="7da8680adb94d704ed6af3231f9619c7.iix" /></p>
<p style="font-family: arial, sans-serif; color: #666666;"> </p>
<p style="font-family: arial, sans-serif; color: #666666;">It&#39;s useful, for instance, when you need to see or modify a field that is not, for whatever reason, displayed on the form view.   You can personalize your list view, add the appropriate fields and update the record if you need to.</p>
<p style="font-family: arial, sans-serif; color: #666666;"> </p>
<p style="font-family: arial, sans-serif; color: #666666;">UI Action details:</p>
<p style="font-family: arial, sans-serif; color: #666666;"><span style="font-weight: inherit; font-style: inherit; font-family: &#39;courier new&#39;, courier;">Name: Switch to List View<br /></span><span style="font-weight: inherit; font-style: inherit; font-family: &#39;courier new&#39;, courier;">Table: Global [global]<br /></span><span style="font-weight: inherit; font-style: inherit; font-family: &#39;courier new&#39;, courier;">Order: 200,000<br /></span><span style="font-weight: inherit; font-family: &#39;courier new&#39;, courier; font-style: inherit;">Active: checked<br /></span><span style="font-weight: inherit; font-style: inherit; font-family: &#39;courier new&#39;, courier;">Show insert: checked<br /></span><span style="font-weight: inherit; font-style: inherit; font-family: &#39;courier new&#39;, courier;">Show update: checked<br /></span><span style="font-weight: inherit; font-style: inherit; font-family: &#39;courier new&#39;, courier;">Client: checked<br /></span><span style="font-weight: inherit; font-style: inherit; font-family: &#39;courier new&#39;, courier;">Form context menu: checked<br /></span><span style="font-family: &#39;courier new&#39;, courier; font-style: inherit; font-weight: inherit;">Hint: Switches to the List View for the table, returning this one record<br /></span><span style="font-weight: inherit; font-style: inherit; font-family: &#39;courier new&#39;, courier;">Onclick: u_switchToListView()<br /></span><span style="font-weight: inherit; font-style: inherit; font-family: &#39;courier new&#39;, courier;">Condition: gs.hasRole(&#34;admin&#34;)<br /></span><span style="font-weight: inherit; font-style: inherit; font-family: &#39;courier new&#39;, courier;">Script:<br /></span></p>

      try{
              window.location &#61; &#34;/&#34; &#43; g_form.getTableName() &#43; &#34;_list.do?sysparm_query&#61;sys_id%3d&#34; &#43; g_form.getUniqueValue();
      } catch(err){}
