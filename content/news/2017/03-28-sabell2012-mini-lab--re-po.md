font-family: arial, sans-serif; color: #666666;"></p><p style="margin-bottom: 0.0001pt; font-family: arial, sans-serif; color: #666666;"><span style="font-size: 12pt;">A short time ago I had need of changing my repository URL for one of my projects in the Studio IDE, and much to my surprise found that I was not allowed to do it!   The URL field was read-only!?!   Off to the ServiceNow documentation site I went.   However, no solution was published there.   I promptly went to the Community to find out if this had been mentioned by anyone else.   Sure enough, the question was asked, and <a title="ctomasi" __default_attr="2218" __jive_macro_name="user" class="jive_macro jive_macro_user" data-orig-content="ctomasi" data-renderedposition="185_616.421875_75_17" href="/community?id=community_user_profile&user=7ae05a61db981fc09c9ffb651f9619a2">ctomasi</a> was able to get an answer (<a title="How do you edit/remove a source control repository?" __default_attr="253435" __jive_macro_name="thread" class="jive_macro_thread jive_macro" data-orig-content="How do you edit/remove a source control repository?" data-renderedposition="185_8_1144_41" href="/community?id=community_question&sys_id=d1f05fa9dbdcdbc01dcaf3231f96194e">How do you edit/remove a source control repository?</a>).   But the answer there bothered me.   1) it was a work-around, 2) you lost all historical information with the work-around.   After my third project re-point, using this method, I began looking around for the best way to hack it (I was that unhappy).   I began investigating all the pieces I could find concerning where the URL might be located, and what I would have to write in the way of a Fix Script to do the actual hack.   </span></p><p style="margin-bottom: 0.0001pt; font-family: arial, sans-serif; color: #666666;"></p><p style="margin-bottom: 0.0001pt; font-family: arial, sans-serif; color: #666666;"><span style="font-size: 12pt;">Well, sometimes the complex way is NOT the solution!   </span></p><p style="margin-bottom: 0.0001pt; font-family: arial, sans-serif; color: #666666;"></p><p style="margin-bottom: 0.0001pt; font-family: arial, sans-serif; color: #666666;"><span style="font-size: 12pt;">While messing around looking at tables and their interactions I tripped across a method for doing this that looks like it was intentional; sic. out-of-the-box.   It was too easy.   This method does not hang onto the Source Control Branch or Source Control Tags (a minor loss to me), but just so you are warned.   </span></p><p style="margin-bottom: 0.0001pt; font-family: arial, sans-serif; color: #666666;"></p><p style="margin-bottom: 0.0001pt; font-family: arial, sans-serif; color: #666666;"><span style="font-size: 12pt;">This article explains how to change the URL using that method.</span></p><p style="margin-bottom: 0.0001pt; font-family: arial, sans-serif; color: #666666;"></p><p style="margin-bottom: 0.0001pt; font-family: arial, sans-serif; color: #666666;"><span style="font-size: 12pt;">So WHY would you want to change the URL in the first place?   Here are some use-cases I can think of just off the top of my head:</span></p><p style="margin-bottom: 0.0001pt; font-family: arial, sans-serif; color: #666666;"></p><ul><li><span style="font-size: 12pt;">Changing from a personal repository to a corporate repository.   My original reason for changing the URL.</span></li><li><span style="font-size: 12pt;">Reorganization of the repository's directory structure requires changing of the Studio url.   This is probably the most important.</span></li><li><span style="font-size: 12pt;">Unbinding source control from the Studio project.</span></li><li><span style="font-size: 12pt;">Changing the repository provider.   Let's say from GitLab to GitHub or the reverse.</span></li></ul><p></p><p><span style="font-size: 12pt;">And I am sure there are more.</span></p><p></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif;"><strong>BEST PRACTICE: First make a backup of your entire project to an Update Set</strong></span></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif;">1. Navigate to System Applications -&gt; Applications. The Applications list view will be displayed.</span></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"><span style="font-family: arial, helvetica, sans-serif;"><span style="font-size: 12pt;">2. Click on the link to your Studio Application. This will display the Custom Application properties form.</span><span style="font-size: 12pt;"> </span></span></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"><img   class="image-9 jive-image" height="362" src="574e8086db909fc03eb27a9e0f96193f.iix" style="max-width: 1200px; max-height: 900px; width: 616px; height: 362.096px;" width="616"/></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif;">3. Change the Current Application setting to be the application you are working with. This will activate the Related Links on the form.</span></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"><img   class="jive-image image-10" height="325" src="0f7aafb9db581fc03eb27a9e0f961970.iix" style="max-width: 1200px; max-height: 900px; width: 630px; height: 324.975px;" width="630"/></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif;">4. Scroll down the form to view the related links and click on the Publish to Update Set… link. The Publish to Update Set</span></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"><img   class="image-11 jive-image" src="4f7d81c6db9413043eb27a9e0f961959.iix" style="max-width: 1200px; max-height: 900px;"/></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif;">5. Fill in the form:</span></p><p></p><p style="padding-left: 30px;"><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif;"><strong>Version</strong>: &lt;&lt;keep what was displayed&gt;&gt;</span></p><p style="padding-left: 30px;"><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif;"><strong>Description</strong>: Repoint to new repository.</span></p><p style="padding-left: 30px;"><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif;">Click on the Publish button.</span></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px; padding-left: 30px;"></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"><img   class="image-12 jive-image" height="295" src="678e2c86db58dfc03eb27a9e0f9619e9.iix" style="max-width: 1200px; max-height: 900px; width: 476px; height: 294.727px;" width="476"/></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif;">6. Wait for the Progress bar to complete and click the Done button. The new update set will be displayed.</span></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif;">7. Under Related Links click the Export to XML button. This will save a copy of the entire project to your local disk.   You might want to rename this with your Application name and version just to be safe.</span></p><p></p><p></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"><span style="font-size: 14pt; font-family: arial, helvetica, sans-serif;"><strong>To Clear out the Repository URL For an Application</strong></span></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"><span style="font-size: 12pt;">Now, on to the good stuff!</span></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif;">1. Navigate to sys_repo_branch.list. The Source Control Branches list view will be displayed.</span></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif;">2. Find the Application you wish to reset the repository URL for, and open that record. The Source Control Branch form will be displayed.</span></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"><img   class="image-13 jive-image" height="190" src="831b840edb5c1304b322f4621f9619fc.iix" style="max-width: 1200px; max-height: 900px; width: 742px; height: 189.828px;" width="742"/></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif;">3. Click on the Repository Configuration link button. The Repository Configuration Record will be displayed.</span></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"><img   class="jive-image image-14" height="177" src="f158a371dbd41fc068c1fb651f96197e.iix" style="max-width: 1200px; max-height: 900px; width: 745px; height: 176.938px;" width="745"/></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif;">4. Click on the Delete button in the upper right corner of the form. The Confirmation form will appear.</span></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif;"><img   class="image-15 jive-image" height="295" src="e55851cadb58130468c1fb651f9619c9.iix" style="max-width: 1200px; max-height: 900px; height: 294px; width: 743.077px;" width="744"/></span></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif;">5. Click on the delete button.</span></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"><span style="font-family: arial, helvetica, sans-serif;"><img   class="image-7 jive-image" height="201" src="1610550edb181b04ed6af3231f9619d0.iix" style="max-width: 1200px; max-height: 900px; width: 390px; height: 200.304px;" width="390"/></span></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif;">6. A "Record Not Found" message will be displayed.   Ignore this, and go back to the Studio to open your application.</span></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"><img   class="image-16 jive-image" src="cdb3b886db589304b322f4621f961952.iix" style="max-width: 1200px; max-height: 900px;"/></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"></p><p style="color: #000000; font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 11px;"><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif;">7. This clears out the repository URL and information for the Application. You may now use the normal linking of a repository procedure inside of Studio; to point at your new URL.</span></p><p></p><p><span style="font-size: 12pt;"><span style="font-family: arial, helvetica, sans-serif;">And there you go!   An out-of-the-box method for clearing and/or changing the repository URL for ServiceNow Studio.   </span>Maybe in a future release we will get a menu-item or editable URL field inside the Studio to do this, but until then this is the best way.</span></p><p><span style="font-size: 12pt;"><br/></span><span style="font-weight: inherit;
