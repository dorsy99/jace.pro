<p>The Jakarta release of ServiceNow had a ton of new functionalities added. You now have the opportunity to <a title="ocs.servicenow.com/bundle/jakarta-servicenow-platform/page/administer/plugins/concept/c_ServiceNowPlugins.html" href="https://docs.servicenow.com/bundle/jakarta-servicenow-platform/page/administer/plugins/concept/c_ServiceNowPlugins.html">enable plugins</a> in this release that you didn't have access to before. Plugins are software components that give you control to enable or disable features and functionalities in your instance. See <a title="ocs.servicenow.com/bundle/jakarta-servicenow-platform/page/administer/plugins/reference/jakarta-list-of-plugins.html" href="https://docs.servicenow.com/bundle/jakarta-servicenow-platform/page/administer/plugins/reference/jakarta-list-of-plugins.html">Jakarta plugins</a> for a fill list of all plugins available in the Jakarta release.</p><p></p><p>Plugins can be turned on a few different ways:</p><ol><li>Active by default in an instance.</li><li><a title="ocs.servicenow.com/bundle/jakarta-servicenow-platform/page/administer/plugins/task/t_ActivateAPlugin.html" href="https://docs.servicenow.com/bundle/jakarta-servicenow-platform/page/administer/plugins/task/t_ActivateAPlugin.html">Manually activated</a> by an admin user.</li><li><a title="ocs.servicenow.com/bundle/jakarta-servicenow-platform/page/administer/plugins/task/t_RequestAPlugin.html" href="https://docs.servicenow.com/bundle/jakarta-servicenow-platform/page/administer/plugins/task/t_RequestAPlugin.html">By request from ServiceNow.</a></li><li><a title="ocs.servicenow.com/bundle/jakarta-servicenow-platform/page/administer/plugins/task/t_PurchaseAPlugin.html" href="https://docs.servicenow.com/bundle/jakarta-servicenow-platform/page/administer/plugins/task/t_PurchaseAPlugin.html">For a price.</a></li></ol><p></p><p>Plugins share a special relationship in each product and across the platform. Many plugins are associated and can be leveraged for enhanced capabilities. The system should notify you if you are trying to use a plugin that requires other plugins to operate successfully.</p><p style="text-align: center;"><img   alt="Plugins--HI-service-catalog-form.jpg" class="image-2 jive-image" src="476b9d86db1cd344e9737a9e0f961973.iix" style="width: 620px; height: 405px;"/></p><pre __jive_macro_name="quote" class="jive_macro_quote jive_text_macro" data-renderedposition="1001_8_269_15"><p>In some cases, a plugin cannot work at all unless one or more other plugins are installed. When you ask to activate a plugin in this category, the system notifies you of the dependencies before it completes the activation.
