<p><span style="font-family: arial, helvetica, sans-serif; font-size: 10pt;">With the Madrid release we are excited to bring you a host of new Now Platform<sup>®</sup> capabilities and features. This post will provide a snapshot of our biggest updates, but for the full list of changes, please see the full <a href="https://docs.servicenow.com/bundle/madrid-release-notes/page/release-notes/now-platform-capabilities/now-platform-capabilities-rn.html" rel="nofollow">Madrid release notes</a>.</span></p>
<h4><span style="font-family: arial, helvetica, sans-serif;"><strong><span style="color: #000000;">ServiceNow<sup>®</sup> Mobile Studio and Mobile Apps</span></strong></span></h4>
<p><span style="font-family: arial, helvetica, sans-serif; font-size: 10pt;">In every release, we strive to give our customers new tools to deliver intuitive and engaging experiences for their users. In keeping with this, we are thrilled to announce <a href="https://docs.servicenow.com/bundle/madrid-release-notes/page/release-notes/mobile/now-mobile-platform-rn.html" rel="nofollow">Mobile Apps and ServiceNow Mobile Studio!</a></span></p>
<p><span style="font-family: arial, helvetica, sans-serif; font-size: 10pt;"><a href="https://docs.servicenow.com/bundle/madrid-mobile/page/administer/tablet-mobile-ui/task/configure-mobile-app-studio.htmlhtml" rel="nofollow">Build and deploy</a> native Android and iOS mobile apps with no-code packaged components. Apps easily integrate with hardware capabilities like camera and location services for consumer-like experiences, and with offline read and write you can take them anywhere. Worried about security? All mobile apps and Now Platform data will remain secure whether you’re at home, on the go, online, or offline.</span></p>
<p><span style="font-family: arial, helvetica, sans-serif; font-size: 10pt;"> Feature Highlights: </span><br /><span style="font-family: arial, helvetica, sans-serif; font-size: 10pt;">     • Rapid App Development </span><br /><span style="font-family: arial, helvetica, sans-serif; font-size: 10pt;">     • Out-of-the-box apps for ITSM &amp; FSM </span><br /><span style="font-family: arial, helvetica, sans-serif; font-size: 10pt;">     • Simple Configurations </span><br /><span style="font-family: arial, helvetica, sans-serif; font-size: 10pt;">     • Enhanced Security </span><br /><span style="font-family: arial, helvetica, sans-serif; font-size: 10pt;">     • Work Anytime, Anywhere</span><br /><br /><br /></p>

<h4><span style="font-family: arial, helvetica, sans-serif;"><strong>Automated Test Framework </strong></span></h4>
<p><span style="font-family: arial, helvetica, sans-serif; font-size: 10pt;">Getting started with Automated Test Framework is now easier than ever. In Madrid, we’re introducing Quick Start Test Frameworks packaged with all ServiceNow products. These pre-packaged tests provide scripts for you to get a jump start on automating regression testing. Out-of-the box scripts can be customized to fit your processes and data. All tests are upgrade safe, so they can run and be re-run on upgrades or changes made to your implementation.</span></p>
<p><span style="font-family: arial, helvetica, sans-serif; font-size: 10pt;">We’ve also made a number of other additions, including:</span></p>
<ul><li><span style="font-family: arial, helvetica, sans-serif; font-size: 10pt;"><a href="https://docs.servicenow.com/bundle/madrid-application-development/page/administer/auto-test-framework/concept/custom-ui-test-steps.html" rel="nofollow">Custom UI Testing Framework</a>: Set up and run automated tests for the functionality and navigability of your custom  user interfaces. Automated UI testing ensures all steps an end user would follow are properly reacting to the user’s input(s). </span></li><li><span style="font-family: arial, helvetica, sans-serif; font-size: 10pt;"><a href="https://docs.servicenow.com/bundle/madrid-application-development/page/administer/auto-test-framework/concept/parameterized-tests.html" rel="nofollow">Parametrized Testing</a>: Speed up your testing by incorporating multiple data values in a single test. With Parametrized Testing, you no longer need to duplicate tests to run separate sets of data values.</span></li></ul>
<p><span style="font-family: arial, helvetica, sans-serif; font-size: 10pt;">Be sure to check out the full release notes of <a href="https://docs.servicenow.com/bundle/madrid-application-development/page/administer/auto-test-framework/concept/automated-test-framework.html" rel="nofollow">Automated Test Framework</a> for more!</span></p>

<h4><span style="font-family: arial, helvetica, sans-serif;"><strong>Antivirus Scanning</strong></span></h4>
<p><span style="font-family: arial, helvetica, sans-serif; font-size: 10pt;">Keep your instances safe with secure file scanning. <a href="https://docs.servicenow.com/bundle/madrid-platform-administration/page/administer/security/concept/antivirus-protection.html" rel="nofollow">Antivirus Scanning</a> helps protect your instances against virus infections that can be introduced via file attachments to your system records, such as incidents, problems, and stories. Turned on across all instances by default, administrators can easily configure and toggle scanning from the instance level down to individual tables. Gain visibility into threats to diagnose problems and increase response times knowing that all infected files are automatically quarantined for review.</span></p>
<p><span style="font-family: arial, helvetica, sans-serif; font-size: 12pt;"><img style="max-width: 100%; max-height: 480px;" src="https://community.servicenow.com/16a0aa1edb772784a39a0b55ca961999.iix" /></span></p>
<h4><span style="font-family: arial, helvetica, sans-serif;"><strong>Usability Enhancements</strong></span></h4>
<p><span style="font-family: arial, helvetica, sans-serif; font-size: 10pt;">Here are a few of our favorite quality of life and usability enhancements in the Madrid release. For the full list, see the <a href="https://docs.servicenow.com/bundle/madrid-release-notes/page/release-notes/now-platform-capabilities/now-platform-capabilities-rn.html" rel="nofollow">Madrid release notes</a>.</span></p>
<p><span style="font-family: arial, helvetica, sans-serif; font-size: 10pt;"><a href="https://docs.servicenow.com/bundle/madrid-platform-administration/page/administer/security/concept/instance-security-center.html" rel="nofollow">Instance Security Center</a></span></p>
<ul><li><span style="font-family: arial, helvetica, sans-serif; font-size: 10pt;">Formerly known as Instance Security Dashboard, Instance Security Center has been redesigned and rebuilt using Service Portal to provide more intuitive insights and user experiences. The homepage offers administrators a succinct snapshot of their current security levels to identify and respond to any security lapses on their instance.</span></li><li><span style="font-family: arial, helvetica, sans-serif; font-size: 10pt;">Four new trackable events have been added to the existing portfolio of events, which can now be customized to an administrator’s use case. Administrators can set the thresholds for “failed logging attempts” and monitor ServiceNow login metrics.</span></li></ul>
<p><span style="font-family: arial, helvetica, sans-serif; font-size: 10pt;"><a href="https://docs.servicenow.com/bundle/madrid-platform-user-interface/page/build/help-guided-tours/concept/guided-tours.html" rel="nofollow">Guided Tour Designer</a></span></p>
<ul><li><span style="font-family: arial, helvetica, sans-serif; font-size: 10pt;">Creating callouts within the Guided Tour flow designer interface has been streamlined down to one “create callout” button. When placing a callout, Guided Tour will now automatically choose the logical direction for the next step, decreasing the margin for error in creating a flow.</span></li><li><span style="font-family: arial, helvetica, sans-serif; font-size: 10pt;">To prevent a tour from accidentally targeting employees of all roles, Guided Tour will now explicitly prompt the tour creator for a role assignment. This disallows the possibility of leaving the role field blank, thus publishing the tour to all employees.</span></li></ul>
<p><span style="font-family: arial, helvetica, sans-serif; font-size: 10pt;"><a href="https://docs.servicenow.com/bundle/madrid-servicenow-platform/page/build/service-portal/concept/c_ServicePortal.html" rel="nofollow">Service Portal</a></span></p>
<ul><li><span style="font-family: arial, helvetica, sans-serif; font-size: 10pt;">Service Portal administrators now have the ability incorporate multiple knowledge bases and catalogs while relying on role- and user-based permissions to limit which articles and catalog items end-users see.</span></li></ul>
<p><span style="font-family: arial, helvetica, sans-serif; font-size: 10pt;"><a href="https://docs.servicenow.com/bundle/madrid-servicenow-platform/page/administer/metricbase/concept/metricbase.html" rel="nofollow">MetricBase</a></span></p>
<ul><li><span style="font-family: arial, helvetica, sans-serif; font-size: 10pt;">You can now migrate data from one series to another via a Java Script API to refactor your data and change the retention policy. Admins have the ability to change the aggregates used for compaction in retention policy, depending on the specific use case of the data set.</span></li><li><span style="font-family: arial, helvetica, sans-serif; font-size: 10pt;">New data analysis tools are available to resample on various transforms using standard deviation, chi-square, and median via a Java script API. Relevant data is now made easier to filter down when performing data analysis.</span></li></ul>
