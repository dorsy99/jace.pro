<p class="p2"><img src="https://community.servicenow.com/13f53b65dbc184105129a851ca961939.iix" /></p>
<p class="p1"><span style="font-family: helvetica; font-size: 12pt;">If you haven’t read my <a href="https://community.servicenow.com/community?id&#61;community_blog&amp;sys_id&#61;64a27f17dbbc08942be0a851ca961911" rel="nofollow"><span class="s1">previous blog</span></a> which introduces you to the basic ways to navigate content on the <a href="https://nowlearning.service-now.com/lxp" rel="nofollow"><span class="s1">Now Learning</span></a> platform please take a minute to get yourself familiar with the site. </span></p>
<p class="p1"><span style="font-family: helvetica; font-size: 12pt;">To start your training journey with ServiceNow it is recommended that you start with the basics.<span class="Apple-converted-space">  </span>We have two courses we recommend you start with, the first is a short On Demand course and the second is a longer instructor-led training.<span class="Apple-converted-space">  </span>Read on to learn more about both of these trainings.</span></p>
<p class="p2"> </p>
<p class="p3"><strong><span style="font-family: helvetica; font-size: 14pt;">Get Started with the Now Platform</span></strong></p>
<p class="p1"><span style="font-family: helvetica; font-size: 12pt;">We are excited to announce the release of a new On Demand course called <a href="https://nowlearning.service-now.com/lxp?id&#61;overview&amp;sys_id&#61;9455b7f91b288050528f639fbd4bcb48&amp;type&#61;course" rel="nofollow"><span class="s1">Get Started with the Now Platform</span></a>.</span></p>
<p class="p1"><span style="font-family: helvetica; font-size: 12pt;">In this free 40-minute online course you will get an overview of the navigation and available features to help you to be more efficient in your day-to-day work while using the Now Platform. The modules within the training will help you to understand the benefits of the Now Platform. You will be able to set up a homepage to display highlights across a user&#39;s records. Learn how to complete and update forms with real-time information and view and organize records. You will learn to communicate and collaborate within the Now Platform as well as collaborate visually on records. Finally, how to create and share reports will be reviewed.</span></p>
<p class="p4"> </p>
<p class="p3"><strong><span style="font-family: helvetica; font-size: 14pt;">ServiceNow Fundamentals</span></strong></p>
<p class="p1"><span style="font-family: helvetica; font-size: 12pt;">Many learners start their ServiceNow training journey with <a href="https://nowlearning.service-now.com/lxp?id&#61;overview&amp;sys_id&#61;e4383a8cdb5eff40de3cdb85ca96190e&amp;type&#61;course" rel="nofollow"><span class="s1">ServiceNow Fundamentals</span></a>.</span></p>
<p class="p1"><span style="font-family: helvetica; font-size: 12pt;">ServiceNow Fundamentals is a three-day course taught both as Instructor-Led Training (ILT) and Virtual Instructor-Led Training (VILT) in time zones throughout the world. Coming soon we are excited to offer this popular course in an On-Demand format! The ServiceNow Fundamentals course summarizes fundamental platform features by describing the ServiceNow user interface, providing a summary of the database schema, demonstrating uses for commonly accessed applications, and introducing advanced functionality, such as scripting and application development.</span></p>
<p class="p1"><span style="font-family: helvetica; font-size: 12pt;">The course weaves a scenario throughout to present real-world, relevant lab exercises. The class features lecture and group discussions, as well as extensive hands-on practice and reinforcement delivered in a variety of labs.</span></p>
<p class="p1"><span style="font-family: helvetica; font-size: 12pt;">You will start with User Interface and Navigation where you will become familiar with how users can access ServiceNow; the tables where Users and Groups are stored; the three components of the Now Platform® User Interface; and how to navigate to applications and modules in ServiceNow using the Application Navigator.</span></p>
<p class="p1"><span style="font-family: helvetica; font-size: 12pt;">Next you will move on to Collaboration and learn about Common tables that extend the task table; two methods of automatically assigning tasks to users; the differences between the My Work and the My Groups Work modules; what information is stored in the Activity Stream for a task; the difference between events and notifications; how dot-walking is used in notification configuration; and how to use Connect Chat to collaborate on a task.</span></p>
<p class="p1"><span style="font-family: helvetica; font-size: 12pt;">After that the course takes you through Database Administration covering topics such as the hierarchy of tables, records, and fields; the classes of tables in ServiceNow; different field types in a record; the tables where users, groups, and roles are stored; how to create access control rules; how access controls are evaluated; and how to use import sets to populate data in a table.</span></p>
<p class="p1"><span style="font-family: helvetica; font-size: 12pt;">Self-Service and Process Automation follows with creating a knowledge base article by importing a Word document; how to locate the articles needing approval; and defining, applying, and testing user criteria on a knowledge base.</span></p>
<p class="p1"><span style="font-family: helvetica; font-size: 12pt;">Then you will get an Introduction to Scripting and Application Tools where you will learn how to differentiate between client-side and server-side scripts; identify the three types of UI Policy Actions; explain the purpose of a Data Policy; and summarize how Integration Hub simplifies 3rd party integrations.</span></p>
<p class="p1"><span style="font-family: helvetica; font-size: 12pt;">Lastly is the Capstone Project where you have the opportunity to practice everything you have learned by completing eight tasks to set up a ServiceNow instance for a fictitious company.</span></p>
