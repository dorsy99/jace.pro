<p>We are thrilled to bring you a host of new Now Platform<strong>®</strong> App Engine features and capabilities in the New York release. This post will provide a snapshot of a few of our favorite updates, but for the full list, please see the full <a href="https://docs.servicenow.com/bundle/newyork-release-notes/page/release-notes/family-release-notes.html" rel="nofollow">New York release notes</a>. </p>
<p> </p>
<h4><strong>Now Mobile</strong> </h4>
<p>With the New York release, we are excited to introduce <a href="https://docs.servicenow.com/bundle/newyork-release-notes/page/release-notes/mobile/mobile-employee-rn.html" rel="nofollow">Now Mobile</a>, our brand-new app for iOS and Android devices that lets employees get work done from anywhere. </p>
<p>The Now Mobile app allows employees — not just agents — to get work done, wherever they are. Manage approvals, open new requests, and view your to-do list on mobile. Global search allows you find employees, catalog items, services, or KB article information from a single location. <a href="https://docs.servicenow.com/bundle/newyork-performance-analytics-and-reporting/page/administer/virtual-agent/concept/virtual-agent-overview.html" rel="nofollow">Virtual Agent</a> is available right within the app and can be even be launched through a Siri Shortcut. </p>
<p>Feature Highlights: </p>
<p>Act on Approvals and Orders </p>
<p>Open Requests and Manage To-Dos </p>
<p>Global Search Helps You Find What You Need Fast </p>
<p>Virtual Agent Assistance Anytime, Anywhere  </p>
<p> </p>
<h4><strong>Guided App Creator</strong> </h4>
<p>With every release, we try to raise the bar of innovation and enhancements that make building workflow apps easier than ever. <a href="https://docs.servicenow.com/bundle/newyork-release-notes/page/release-notes/application-development/guided-app-creator-rn.html" rel="nofollow">Guided App Creator</a> enables customers to quickly and effortlessly create new applications.  </p>
<p>Within a few clicks, users can transform their Excel spreadsheets into data tables, enabling them to harness the power of the platform. Through this new guided experience, users can stand up a new Workspace and/or build a <a href="https://docs.servicenow.com/bundle/newyork-mobile/page/administer/tablet-mobile-ui/concept/mobile-studio.html" rel="nofollow">mobile application</a>, allowing them to take their data on the go.  </p>
<p>Feature highlights: </p>
<p>Rapid App Development </p>
<p>Guided Support for No Code Builders </p>
<p>Easy Spreadsheet to Table Converter </p>
<p>Work on the Go with a Mobile App  </p>
<p>  </p>
<h4><strong>Pro Dev Tools</strong> </h4>
<p>In New York, we continue to build upon our world-class developer experience by providing even more <a href="https://docs.servicenow.com/bundle/newyork-application-development/page/build/applications/concept/c_ApplicationTools.html" rel="nofollow">robust tooling</a>.  We’ve added Intellisense for HTML and jelly tags, code-formatting, and ES-Lint support, all of which help developers write the best possible code. Within the Debugger, we’ve also added support for conditional breakpoints and session logs, making it even easier to diagnose and troubleshoot issues on the platform. </p>
<p>Deep Dive: </p>
<p>Intellisense helps developers understand the objects and APIs they can use while coding, making for faster application development. </p>
<p>Conditional break-points in the Debugger allows developers to improve their ability to troubleshoot their code during testing and debugging. </p>
<p>Script editor to adopt <a href="https://docs.servicenow.com/bundle/newyork-application-development/page/script/general-scripting/concept/monaco-editor.html" rel="nofollow">Monaco control</a> for all new UIs, and move feature set closer to parity to other development platforms. </p>
<p>To find out more about Pro Dev Tools, please visit the <a href="https://docs.servicenow.com/bundle/newyork-release-notes/page/release-notes/family-release-notes.html" rel="nofollow">New York release notes</a>. </p>
<p> </p>
<h4><strong>Usability Enhancements</strong> </h4>
<p>Each release we make enhancements to our favorite tools for the <a href="https://docs.servicenow.com/bundle/newyork-release-notes/page/release-notes/application-development/application-development-rn.html" rel="nofollow">Now Platform<strong>®</strong> App Engine</a>. Here are a few highlights, but please see the <a href="https://docs.servicenow.com/bundle/newyork-release-notes/page/release-notes/family-release-notes.html" rel="nofollow">New York release notes</a> for a full list. </p>
<p> </p>
<p><strong><a href="https://docs.servicenow.com/bundle/newyork-release-notes/page/release-notes/application-development/automated-test-framework-rn.html" rel="nofollow">Automated Test Framework</a> </strong></p>
<p>Automated Test Framework eliminates the #1 barrier to upgrading and staying current – manual regression testing. </p>
<ul><li>Automate regression testing with a growing library of quick start tests. In New York, we added 120 new tests that expand across workflows in IT, Security, HR, and CSM. </li><li>Accelerate testing by running multiple tests in parallel. To avoid potential conflicts ATF will automatically identify conflicting tests and run them in the appropriate sequence. </li><li>Upgrade faster with high speed patch update time from tens of minutes to as fast as seconds. </li></ul>
<p> </p>
<p><strong><a href="https://docs.servicenow.com/bundle/newyork-release-notes/page/release-notes/now-platform-capabilities/service-portal-rn.html" rel="nofollow">Service Portal</a> </strong></p>
<p>Search easier with <a href="https://docs.servicenow.com/bundle/newyork-servicenow-platform/page/build/service-portal/task/configure-va-in-sp.html" rel="nofollow">Agent Chat</a>, plus, recommended search options and embedded links to KB articles within related requests. </p>
<ul><li>Using machine-learning and Agent Chat, we enabled Service Portal to help customers search for their needs faster and easier through recommended search options and on-demand support. </li><li>Enable efficient self-service with embedded KB articles within related requests. </li></ul>
<p> </p>
<p><strong><a href="https://docs.servicenow.com/bundle/newyork-release-notes/page/release-notes/now-platform-capabilities/visual-task-boards-rn.html" rel="nofollow">Visual Task Boards </a></strong></p>
<p>Prioritize efficiently by dividing tasks both vertically and horizontally. Add a capacity limit on Work in Progress vertical lanes. </p>
<ul><li>Add swim lanes (aka horizontal lines) to any board to model a 2nd dimension. E.g. incident types </li><li>Set a capacity limit on any vertical lane with the Work in Progress Limit. </li><li>Refreshed the UI by removing unnecessary visual load, reducing space reserved for quick filters, and adding new color themes. </li></ul>
