<h3>IntegrationHub has arrived in New York</h3>
<p><span style="font-size: 12pt;">Natively integrate ServiceNow with anything</span></p>
<p style="text-align: center;"><img style="max-width: 100%; max-height: 480px;" src="https://community.servicenow.com/22cb5e6fdb67ffc414d6fb243996197a.iix" width="551" height="296" /></p>
<p style="text-align: left;"> New York delivers enterprise grade integrations with significant innovations to enable developers and IT generalists to create powerful reusable integrations and components. And the library of codeless connectors – called Spokes - in the <a href="https://store.servicenow.com/sn_appstore_store.do#!/store/search?category&#61;IntegrationHub&amp;orderby&#61;rating" rel="nofollow">ServiceNow Store</a> continues to grow, empowering everyone including no-code process analysts to easily extend digital workflows to 3rd party services. For the full list of changes, please see the <a href="https://docs.servicenow.com/bundle/newyork-release-notes/page/release-notes/now-platform-capabilities/integrationhub-rn.html" rel="nofollow">New York release notes</a>.</p>
<p>IntegrationHub enables IT to meet integration demand at scale through a publish-discover-re-use architecture and a single place to find integrations and components that boosts app dev and process automation speed and productivity, and accelerates time to value. Developers are empowered to create and share custom integrations and components with IT generalists and business builders. ServiceNow publishes new out-of-the box integrations every month in the ServiceNow store, re-usable by anyone.</p>
<p style="text-align: center;"><img style="max-width: 100%; max-height: 480px;" src="https://community.servicenow.com/c1143ad7db93fb001cd8a345ca9619dd.iix" width="493" height="233" /></p>
<h4><span style="font-size: 12pt;"><strong>New IntegrationHub Features in New York</strong></span></h4>
<p><span style="font-size: 12pt;"><strong>Key Enhancements:</strong></span></p>
<p style="padding-left: 30px;">• Dynamic Inputs with API Introspection <br />• Complex Data <br />• SSH, JDBC, OpenAPI <br />• Data Streaming and Pagination <br />• Retry Policy <br />• No code XML parser <br />• Remote Tables<br />• New out-of-the-box integrations for no-code builders <br />• Discover Store Content</p>
<p> </p>
<h4><strong><a href="https://docs.servicenow.com/bundle/newyork-servicenow-platform/page/administer/flow-designer/concept/dynamic-inputs.html" rel="nofollow">Dynamic Inputs</a></strong></h4>
<p>With Dynamic Inputs you can create integrations that derive their inputs from dynamic external sources like Salesforce, OData, CRMs, ERPs, and more with introspection APIs for table schema and custom objects.</p>
<p style="text-align: center;"><img style="max-width: 100%; max-height: 480px;" src="https://community.servicenow.com/bef768091b0bffc4ada243f6fe4bcb80.iix" width="479" height="270" /></p>
<p style="text-align: left;"> </p>
<h4><strong><a href="https://docs.servicenow.com/bundle/newyork-servicenow-platform/page/administer/flow-designer/concept/complex-data.html" rel="nofollow">Complex Data</a></strong></h4>
<p>Harness complex data when defining Action Inputs, Outputs, UX Experience baseline for Parsing, and For-loops to create custom integrations for scaling business demand.</p>
<p style="text-align: center;"><img style="max-width: 100%; max-height: 480px;" src="https://community.servicenow.com/a9fb9663dba7ffc414d6fb2439961943.iix" width="399" height="300" /></p>
<p style="text-align: left;"> </p>
<h4><strong>SSH, JDBC, and OpenAPI</strong></h4>
<p>Author and pre-configure an Action REST Step from an <a href="https://docs.servicenow.com/bundle/newyork-servicenow-platform/page/administer/integrationhub/concept/open-api-integration.html" rel="nofollow">OpenAPI spec</a>, execute <a href="https://docs.servicenow.com/bundle/newyork-servicenow-platform/page/administer/flow-designer/reference/ssh-step-action-designer.html" target="_blank" rel="noopener noreferrer nofollow">SSH commands</a>, and interact with databases with <a href="https://docs.servicenow.com/bundle/newyork-servicenow-platform/page/administer/flow-designer/reference/jdbc-step.html" target="_blank" rel="noopener noreferrer nofollow">new protocols</a>.</p>
<p style="text-align: center;"><img style="max-width: 100%; max-height: 480px;" src="https://community.servicenow.com/1742fa4edb33734c14d6fb2439961909.iix" width="531" height="253" /></p>
<p style="text-align: center;"> </p>
<h4><a href="https://docs.servicenow.com/bundle/newyork-servicenow-platform/page/administer/integrationhub/concept/data-stream-actions.html" rel="nofollow"><strong>Data Streaming and Pagination</strong></a></h4>
<p>Send REST or SOAP requests from Flow Designer to APIs that return a stream of response data larger than 10 MB, or that return paginated results.</p>
<p style="text-align: center;"><img style="max-width: 100%; max-height: 480px;" src="https://community.servicenow.com/d07c1e27dba7ffc414d6fb2439961999.iix" width="459" height="186" /></p>
<p style="text-align: center;"> </p>
<h4><a href="https://docs.servicenow.com/bundle/newyork-servicenow-platform/page/administer/flow-designer/concept/retry-policy.html" rel="nofollow"><strong>Retry Policy</strong></a></h4>
<p>Leverage default policies or define your own conditions to retry failed requests, so actions don’t fail when a transient network or service error occurs.</p>
<p style="text-align: center;"><img style="max-width: 100%; max-height: 480px;" src="https://community.servicenow.com/3e9c5a67dba7ffc414d6fb2439961935.iix" width="444" height="177" /></p>
<p style="text-align: center;"> </p>
<h4><a href="https://docs.servicenow.com/bundle/newyork-servicenow-platform/page/administer/flow-designer/reference/xml-parser-step-action-designer.html" target="_blank" rel="noopener noreferrer nofollow"><strong>No Code XML Parser</strong> </a></h4>
<p>Parse and identify XML and JSON responses from API calls without having to write script.</p>
<p style="text-align: center;"><img style="max-width: 100%; max-height: 480px;" src="https://community.servicenow.com/57bc1ea7dba7ffc414d6fb2439961900.iix" width="509" height="273" /></p>
<h4 style="text-align: left;"><strong><a href="https://docs.servicenow.com/bundle/newyork-release-notes/page/release-notes/now-platform-capabilities/remote-tables-rn.html" rel="nofollow">Remote Tables</a><br /></strong></h4>
<p style="text-align: left;">Visualize external data in local tables on your instance. View data from external systems such as customer details from CRM as a related list in an account record. Surface key read-only information in a unified view to prevent swivel chair issues.<strong><br /></strong></p>
<p style="text-align: center;"><img style="max-width: 100%; max-height: 480px;" src="https://community.servicenow.com/dda5a4addb5ffb44feb1a851ca961925.iix" width="458" height="360" /> </p>
<h4 style="text-align: left;"><strong>New out-of-the box integrations</strong></h4>
<p style="text-align: left;">An ever increasing supply of ready-to-use integrations for popular business apps published monthly in the <a href="https://store.servicenow.com/sn_appstore_store.do#!/store/search?category&#61;IntegrationHub&amp;orderby&#61;rating" target="_blank" rel="noopener noreferrer nofollow">ServiceNow Store</a> simplify and accelerate integrations for everyone. By the September New York release timeframe, the following new spokes are planned for availability:</p>
<p style="padding-left: 30px;">• Twitter <br />• MS Exchange Server <br />• MS Exchange Online <br />• Google Drive <br />• Microsoft Dynamics 365 <br />• Salesforce <br />• Adobe Sign <br />• Twilio <br />• Zoom <br />• Coupa <br />• Jira v2</p>
<p style="padding-left: 30px; text-align: center;"><img style="max-width: 100%; max-height: 480px;" src="https://community.servicenow.com/8d2d9a6bdba7ffc414d6fb2439961923.iix" width="533" height="278" /></p>
<p style="text-align: center;"> </p>
<h4><a href="https://docs.servicenow.com/bundle/newyork-servicenow-platform/page/administer/flow-designer/task/create-flow.html" target="_blank" rel="noopener noreferrer nofollow"><strong>Discover Store Content</strong></a></h4>
<p>Quickly add new integrations directly into your flows. Discover uninstalled integration spokes published on the ServiceNow store directly via the Action Picker in Flow Designer. </p>
<p style="text-align: center;"><img style="max-width: 100%; max-height: 480px;" src="https://community.servicenow.com/4b3968011b4bffc4ada243f6fe4bcbdf.iix" width="459" height="224" /> </p>
<p style="text-align: center;"> </p>
<h4><strong>Available since Madrid: Write Once, Run Anywhere</strong></h4>
<p>Developers can build integration actions once and call them from anywhere in the platform. In addition to Flow Designer Integrations actions can be called via Action APIs from script includes, business rules, UI Actions, and more.</p>
<p style="text-align: center;"><strong><img style="max-width: 100%; max-height: 480px;" src="https://community.servicenow.com/fab9a4c11b4bffc4ada243f6fe4bcb10.iix" width="363" height="311" /></strong></p>
