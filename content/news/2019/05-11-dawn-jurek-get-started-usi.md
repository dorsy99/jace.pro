<h3 class="ng-scope">How is the ServiceNow Agent mobile app different from the ServiceNow Classic mobile app?</h3>
<p class="ng-scope">ServiceNow Agent mobile app is a native app, and ServiceNow Classic is a hybrid app, which includes both native and web components. Native apps are faster and can leverage the functionality of mobile devices, including:</p>
<ul><li>Sending push notifications and emails</li><li>Making phone calls</li><li>Pulling up locations on a map</li><li>Navigating directly to a URL</li><li>Working offline</li></ul>
<h3 class="ng-scope">Can I work on the mobile app without internet connectivity?</h3>
<p>Yes you can! By planning ahead, you can use offline mode to work offline, then upload your changes when you can connect again. To see how this works, check out our video below:</p>
