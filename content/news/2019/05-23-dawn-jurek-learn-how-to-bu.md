<p>You have this great idea for a mobile app that runs on the ServiceNow platform. How do you bring your vision to life? Now you can build your own mobile apps for the ServiceNow Agent mobile app, starting with the Madrid release. And if you&#39;re ready to dive in, we&#39;ve created videos and training to get you started.</p>
<p>But first, we&#39;ll answer a few FAQs:</p>
<h3>Where can I learn more about the ServiceNow Agent mobile app?</h3>
<p>See our post: <a title="Get started using the ServiceNow Agent mobile app" href="https://community.servicenow.com/community?id&#61;community_blog&amp;sys_id&#61;cb97670edb99f74c2be0a851ca9619f3" target="_blank" rel="noopener noreferrer nofollow">Get started using the ServiceNow Agent mobile app</a> for FAQs and videos to get you familiar with the app.</p>
<h3>Who can build mobile apps?</h3>
<p>Any user with the admin role can build apps.</p>
<h3>Where do I build mobile apps?</h3>
<p>ServiceNow Mobile Studio, the integrated development environment (IDE) within the platform, provides you with the core components you need to build your apps. There are some advanced configurations that you can enable within the platform. See <a title="Mobile app configuration in the platform" href="https://docs.servicenow.com/bundle/madrid-mobile/page/administer/tablet-mobile-ui/concept/mobile-app-configuration-platform.html" target="_blank" rel="noopener noreferrer nofollow">Mobile app configuration in the platform</a> for more information.</p>
