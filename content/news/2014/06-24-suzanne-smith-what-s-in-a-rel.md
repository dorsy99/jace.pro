<p>Do you need information about a specific ServiceNow release, patch, or hotfix? Check out the <span style="text-decoration: underline;"><a class="jive-link-external-small" href="http://wiki.servicenow.com/index.php?title=Release_Notes" rel="nofollow" target="_blank"><span style="color: #0000ff; text-decoration: underline;">Release Notes</span></a></span> on the ServiceNow wiki. For example, the notes for our most recent release, named <span style="text-decoration: underline;"><a class="jive-link-external-small" href="http://wiki.servicenow.com/index.php?title=Eureka_Release_Notes" rel="nofollow" target="_blank"><span style="color: #0000ff; text-decoration: underline;">Eureka</span></a></span>, contain descriptions of all the features and notable changes in the release. You'll find the following information in the release notes:</p><p style="min-height: 8pt; height: 8pt; padding: 0px;">  </p><ul><li>features</li><li>fixes</li><li>versions in the release family</li><li>tips for upgrading</li></ul><p style="min-height: 8pt; height: 8pt; padding: 0px;">  </p><p><a _jive_internal="true" href="/servlet/JiveServlet/showImage/38-3198-11093/release_notes_blog_post.png"><img  alt="release_notes_blog_post.png" class="image-0 jive-image" height="387" src="010c7442db509f048c8ef4621f961956.iix" style="height: auto;" width="565"/></a></p><p style="min-height: 8pt; height: 8pt; padding: 0px;">  </p><p>As patches and hotfixes are released, they have their own page listing just the changes included. For example, <span style="text-decoration: underline;"><a class="jive-link-external-small" href="http://wiki.servicenow.com/index.php?title=Release:Eureka_Patch_1_Hotfix_2" rel="nofollow" target="_blank"><span style="color: #0000ff; text-decoration: underline;">Eureka Patch 1 Hotflix 2</span></a></span> contains a fix for one workflow issue. All patches and hotfixes are cumulative - they contain earlier patches and hotfixes in the same family.</p><p style="min-height: 8pt; height: 8pt; padding: 0px;">  </p><p>Release notes are also available for previous versions and can be especially useful if you are thinking about upgrading. For example, say you are using the Calgary release, but are interested in upgrading to the Dublin release. Do your research by reading the release notes for Dublin to find out if it contains the features and fixes you need.   If it doesn't, check out the Eureka release notes and consider upgrading to Eureka instead.</p><p style="min-height: 8pt; height: 8pt; padding: 0px;">  </p><p>If you have questions about any release notes page or would like to leave a comment, click <strong>Feedback</strong> near the top of the page.</p><p style="min-height: 8pt; height: 8pt; padding: 0px;">  </p><p><a _jive_internal="true" href="/servlet/JiveServlet/showImage/38-3198-11101/release_notes2_blog_post.png"><img  alt="release_notes2_blog_post.png" class="image-0 jive-image" height="115" src="2a6dbbb9db185704ed6af3231f9619f0.iix" style="height: auto;" width="474"/></a></p>
