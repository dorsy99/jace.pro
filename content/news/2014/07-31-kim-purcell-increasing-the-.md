<p>Do you need to export a large set of data to an Excel or XML file? Are you perplexed why the platform stubbornly allows only a certain amount of data for export? The reason for this is intentional; the platform sets a default limit to prevent performance issues caused by large exports. Here are some format-specific export limits:</p><p><a _jive_internal="true" href="/servlet/JiveServlet/showImage/38-3288-12175/Screen Shot 2014-07-30 at 1.27.46 PM.JPG"><img  alt="Screen Shot 2014-07-30 at 1.27.46 PM.JPG" class="image-0 jive-image" height="218" src="9dfe0906db54d344e9737a9e0f96199e.iix" style="height: auto;" width="576"/></a></p><p>You'll notice if you try to export data from a list that exceeds this limit (for example, more than 10,000 records to Excel), a dialog box appears indicating that your records are being exported, but only until it reaches the default limit:</p><p><a _jive_internal="true" href="/servlet/JiveServlet/showImage/38-3288-12176/Screen Shot 2014-07-30 at 1.32.35 PM.JPG"><img  alt="Screen Shot 2014-07-30 at 1.32.35 PM.JPG" class="image-1 jive-image" height="160" src="db18e48edb9457049c9ffb651f9619a5.iix" style="height: auto; display: block; margin-left: auto; margin-right: auto;" width="308"/></a></p><p>In cases where you absolutely need to export a large chunk of data at once, you can create a system property to increase the limit:</p><p style="min-height: 8pt; height: 8pt; padding: 0px;">  </p><ol><li>Enter <span style="font-family: 'courier new', courier;">sys_properties.list</span> into the <strong>filter text</strong> field.</li><li>Click <strong>New</strong>.</li><li>In the <strong>Name</strong> field, enter <span style="font-family: 'courier new', courier;">glide.ui.export.limit.</span></li><li>For type, select <strong>integer</strong>.</li><li>In the <strong>Value</strong> field, enter &lt;new amount&gt;.</li><li>Click <strong>Submit</strong>.</li></ol><p style="min-height: 8pt; height: 8pt; padding: 0px;">  </p><p>You can now export all of your records (for this one, the limit was increased to 20000):</p><p style="min-height: 8pt; height: 8pt; padding: 0px;">  </p><p><a _jive_internal="true" href="/servlet/JiveServlet/showImage/38-3288-12183/Screen Shot 2014-07-30 at 2.17.17 PM.JPG"><img  alt="Screen Shot 2014-07-30 at 2.17.17 PM.JPG" class="jive-image image-2" height="142" src="9d4a80c2db5017041dcaf3231f96190b.iix" style="height: auto; display: block; margin-left: auto; margin-right: auto;" width="306"/></a></p><p>Although this property can be set to any value, exceeding the default export limit can impact system performance. To save resources, another option is to break up the data into smaller sets. See <a title="k-external-small" class="jive-link-external-small" href="http://wiki.servicenow.com/index.php?title=Exporting_Data#Breaking_Up_Large_Exports" rel="nofollow" target="_blank">Breaking Up Large Exports</a> for more information.</p><p style="min-height: 8pt; height: 8pt; padding: 0px;">  </p><p>See <a title="k-external-small" class="jive-link-external-small" href="http://wiki.servicenow.com/index.php?title=Exporting_Data#Export_Limits" rel="nofollow" target="_blank">Export Limits </a>and <a title="k-external-small" class="jive-link-external-small" href="https://hi.service-now.com/kb_view.do?sysparm_article=KB0538324" rel="nofollow" target="_blank">Troubleshooting Exports</a> for additional resources.</p><p>To watch the YouTube video, see <a title="k-external-small" class="jive-link-external-small" href="https://www.youtube.com/watch?v=jZna_EtZFZ4" rel="nofollow" target="_blank">Troubleshooting ServiceNow Export Limits</a>.</p>
