<p style="text-align: center;"><em>"We keep moving forward, opening new doors, and doing new things, because we're curious and curiosity keeps leading us down new paths."</em></p><p style="text-align: center;">-Walt Disney</p><p style="min-height: 8pt; height: 8pt; padding: 0px;">  </p><p>The application navigator is like a door to your instance. Look through that door located on the left-side of the ServiceNow user interface and all the applications and modules you want to make available to your users are there. You have control over what is visible and who can see the visible items.</p><p><a _jive_internal="true" href="/servlet/JiveServlet/showImage/38-3558-14931/applications_modules.jpg"><img  alt="applications_modules.jpg" class="image-0 jive-image" height="538" src="9b8e0086db1417041dcaf3231f96193b.iix" style="height: auto; display: block; margin-left: auto; margin-right: auto;" width="205"/></a></p><p style="min-height: 8pt; height: 8pt; padding: 0px;">  </p><p>Here are some of the basics for editing the applications and modules in your application navigator.</p><p style="min-height: 8pt; height: 8pt; padding: 0px;">  </p><h3>Hiding and Showing Applications</h3><p style="min-height: 8pt; height: 8pt; padding: 0px;">  </p><p>You can show or hide applications as you see fit. Navigate to <strong>System Definition &gt; Application Menus </strong>and click <strong>All</strong> in the breadcrumbs to show all visible and hidden applications. Click an application <strong>Title</strong> and then select or clear the <strong>Active</strong> option (the application navigator refreshes automatically).</p><p style="min-height: 8pt; height: 8pt; padding: 0px;">  </p><p><a _jive_internal="true" href="/servlet/JiveServlet/showImage/38-3558-14754/applications_modules2.png"><img  alt="applications_modules2.png" class="jive-image image-2" height="327" src="dbfee771db98dfc0b322f4621f9619c0.iix" style="height: 309px; width: 620px;" width="656"/></a></p><p style="min-height: 8pt; height: 8pt; padding: 0px;">  </p><h3><strong><br/></strong></h3><h3><strong><br/></strong></h3><h3><strong>Hiding and Showing Application Modules</strong></h3><p style="min-height: 8pt; height: 8pt; padding: 0px;">  </p><p>You can also show or hide modules listed under applications. In the application navigator, right-click an application label and select <strong>Edit Application</strong>. In the <strong>Modules</strong> related list, click a module <strong>Title</strong>. Select or clear the <strong>Active</strong> option (again, the application navigator refreshes automatically).</p><p style="min-height: 8pt; height: 8pt; padding: 0px;">  </p><p><a _jive_internal="true" href="/servlet/JiveServlet/showImage/38-3558-14753/applications_modules3.png"><img  alt="applications_modules3.png" class="image-1 jive-image" height="353" src="3c601c4edbd09fc03eb27a9e0f96194e.iix" style="height: 339px; width: 620px;" width="645"/></a></p><p style="min-height: 8pt; height: 8pt; padding: 0px;">  </p><h3></h3><h3></h3><h3>Controlling Application Visibility Using Roles</h3><p style="min-height: 8pt; height: 8pt; padding: 0px;">  </p><p>Restrict an application so it is visible to only certain roles by navigating to <strong>System Definition &gt; Application Menus, </strong>clicking an application <span style="font-size: 10pt; line-height: 1.5em;"><strong>Title</strong>, unlocking the <strong>Roles</strong> field, and editing the list. Note that currently you can only specify the roles that are able to see an application; you cannot specify roles that should not see the application.</span></p><p><a _jive_internal="true" href="/servlet/JiveServlet/showImage/38-3558-14755/applications_modules4.png"><img  alt="applications_modules4.png" class="jive-image image-3" height="287" src="4bc095c6dbdc97041dcaf3231f9619d1.iix" style="height: auto;" width="565"/></a></p><p style="min-height: 8pt; height: 8pt; padding: 0px;">  </p><p>That covers some of the basics.
