<p style="text-align: center;"><em>"There is no harm in repeating a good thing."</em></p><p style="text-align: center;">-Plato</p><p></p><p>My blog post last week about <a title="" _jive_internal="true" href="/community?id=community_blog&sys_id=684ea2addbd0dbc01dcaf3231f9619fb">viewing your visual task boards</a> seemed to be quite popular, so, in the spirit of Plato, I'd like to bring you another post about the same subject. Let's talk about changing and customizing the appearance of visual task boards.</p><p></p><p>There is a lot of magic in the Board Configuration icon in the upper right corner of a visual task board:</p><p><img   alt="visual_task_boards_2ndpart1.png" class="image-0 jive-image" height="194" src="120d33f5db501344e9737a9e0f96195a.iix" style="height: 193.574193548387px; width: 577px; display: block; margin-left: auto; margin-right: auto;" width="577"/></p><p>Click the icon to see the options:</p><p><img   alt="visual_task_boards_2ndpart2.png" class="image-1 jive-image" height="385" src="de6c19c2dbd8130468c1fb651f961929.iix" style="height: 385px; width: 214.008379888268px; display: block; margin-left: auto; margin-right: auto;" width="214"/></p><p></p><p><strong>Background color</strong></p><p><span style="font-size: 10pt; line-height: 1.5em;">Click any of the colored squares to apply the background color and see how your board will look. Currently there are twelve available colors. All board members can see the background color.</span></p><p></p><p><strong>Switches</strong></p><p><span style="font-size: 10pt; line-height: 1.5em;">For the view and label options, click the switch to toggle an option on or off. In the screenshot below, the </span><span style="font-size: 10pt; line-height: 1.5em;"><strong>Show Labels</strong></span><span style="font-size: 10pt; line-height: 1.5em;"> option is turned on and the </span><span style="font-size: 10pt; line-height: 1.5em;"><strong>Label Names</strong></span><span style="font-size: 10pt; line-height: 1.5em;"> option is turned off (think "green means go!").</span></p><p><img   alt="visual_task_boards_2ndpart3.png" class="image-0 jive-image" height="85" src="51e7f882dbd097041dcaf3231f9619be.iix" style="height: 84.9130434782609px; width: 279px; display: block; margin-left: auto; margin-right: auto;" width="279"/></p><p></p><p><strong>Configure View</strong></p><p>Any changes you make to the view options can only be seen by you, not other board members. <span style="font-size: 10pt; line-height: 1.5em;">You have several view options:</span></p><ul><li><strong>Compact Lanes:</strong> Decreases lane width. This is useful if you have a good number of lanes and want as many lanes as possible to fit on the screen without having to scroll horizontally.</li><li><strong>Compact Tasks:</strong> Hides task details and decreases card vertical length. This is useful if you have many cards in a lane and want to view as many as possible without having to scroll vertically.</li><li><strong>Cover Image:</strong> Displays the first image attached to that card as a cover image. This is an easy way to quickly see if a card has supplemental screenshots or other visuals attached. <br/><img   alt="visual_task_boards_2ndpart4.png" class="image-0 jive-image" height="365" src="804bfc4edb9cdfc068c1fb651f961948.iix" style="height: 365px; width: 215.109717868339px; display: block; margin-left: auto; margin-right: auto;" width="215"/></li><li><strong>Previous Activity:</strong> Displays the Activity stream showing the previous value for values that changed. This can give you a quick idea of what is happening with a task. For example, in the screenshot below, the <strong>Incident state</strong> changed from <strong>Closed</strong> to <strong>Active</strong>, the <strong>Priority</strong> changed from <strong>High</strong> to <strong>Critical</strong>, and the <strong>State</strong> changed from <strong>Closed</strong> to <strong>Active</strong>. This incident is hot.<br/><img   alt="visual_task_boards_2ndpart5.png" class="jive-image image-2" height="140" src="4259d04adb945344e9737a9e0f961981.iix" style="height: 140px; width: 247.641509433962px; display: block; margin-left: auto; margin-right: auto;" width="248"/></li><li><strong>Member Names:</strong> Displays board member names next to their avatars in the <strong>Labels &amp; Members</strong> toolbar. This is useful if board members insist on using comic book characters as their avatar instead of photos of themselves.<br/><img   alt="visual_task_boards_2ndpart8.png" class="image-0 jive-image" src="d2c41402db589704ed6af3231f961913.iix" style="max-width: 620px; height: auto; display: block; margin-left: auto; margin-right: auto;"/></li></ul><h2></h2><h2></h2><p><strong>Labels</strong></p><p><span style="font-size: 10pt; line-height: 1.5em;">Labels color-code and organize cards on the board. The label names and colors appear on the <strong>Labels &amp; Members</strong> toolbar. Colored dots representing the labels appear on cards. For example, in the screenshot below, the <strong>Unknown source of SAP outage</strong> card is labeled with a blue and a red dot indicating that it is a high priority defect.</span></p><p><span style="font-size: 10pt; line-height: 1.5em;"><img   alt="visual_task_boards_2ndpart6.png" class="image-0 jive-image" height="219" src="75a6b331db905fc03eb27a9e0f9619c2.iix" style="height: 219.290322580645px; width: 515px; display: block; margin-left: auto; margin-right: auto;" width="515"/><br/></span></p><p><span style="font-size: 10pt; line-height: 1.5em;">Any configuration changes you make to labels can be seen by all board members. </span><span style="font-size: 10pt; line-height: 1.5em;">There are several label options:</span></p><ul><li><span style="font-size: 10pt; line-height: 1.5em;"><strong>Show Labels</strong>: Displays labels on cards and in the <strong>Labels &amp; Members</strong> toolbar. This is useful if you want to organize and group cards.</span></li><li><span style="font-size: 10pt; line-height: 1.5em;"><strong>Label Names</strong>: Displays the label name next to the label icon in the<strong> Labels &amp; Members</strong> toolbar. The names define the color-coded circles for board members. </span></li><li><span style="font-size: 10pt; line-height: 1.5em;">Edit label text by double-clicking a label name and typing a new name.<br/><img   alt="visual_task_boards_2ndpart7.png" class="image-1 jive-image" height="181" src="84e815cedb58130468c1fb651f961908.iix" style="height: 181.159420289855px; width: 250px; display: block; margin-left: auto; margin-right: auto;" width="250"/><br/></span></li><li><span style="font-size: 10pt; line-height: 1.5em;">Enable or Disable labels by clicking the circle next to a label name. Circles containing a check mark are enabled. At this time, there are five labels to work with. You can disable some if you need less than five. We have had requests for making more than five labels available and that may happen in a future release.</span></li></ul><p><span style="font-size: 10pt; line-height: 1.5em;"><br/></span></p><p><span style="font-size: 10pt; line-height: 1.5em;">This has been a quick tour through the look and feel options for visual task boards. There is more information about visual task boards in the product documentation:</span></p><p></p><p><a href="http://wiki.servicenow.com/index.php?title=Visual_Task_Boards" title="http://wiki.servicenow.com/index.php?title=Visual_Task_Boards"><span style="color: #0000ff; text-decoration: underline;">Visual Task Boards</span></a></p><p><a href="http://wiki.servicenow.com/index.php?title=Using_Visual_Task_Boards" title="http://wiki.servicenow.com/index.php?title=Using_Visual_Task_Boards"><span style="color: #0000ff; text-decoration: underline;">Using Visual Task Boards</span></a></p><p><span style="font-size: 10pt; line-height: 1.5em;"><br/></span></p>
