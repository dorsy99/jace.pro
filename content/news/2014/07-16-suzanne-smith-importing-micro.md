<p>The ServiceNow Project application is a tool for managing large and small projects. The application includes visual aids such as Gantt charts and work breakdown structure lists. Project also integrates with other ServiceNow applications including Change and Reports.</p><p style="min-height: 8pt; height: 8pt; padding: 0px;">  </p><p>If you are a Microsoft Project user and want to try the ServiceNow Project application, get a fast start by importing a Microsoft Project 2003 or later file into ServiceNow. You can import the file as a new project or import it into an existing ServiceNow Project using the following steps:<br/><br/></p><ol style="list-style-image: none; margin-top: 0.3em; margin-left: 3.2em; color: #333333; font-family: Omnes-pro, Arial, Verdana, sans-serif; text-align: -webkit-auto;"><li>Navigate to <strong>Project &gt; Administration &gt; Import Project</strong>.</li><li>Under <strong>Choose project to import</strong>, click <strong>Browse</strong> and find the Microsoft Project file.</li><li>Do one of the following:<ul><li style="margin-bottom: 0.1em;">To import the Microsoft project as a new project, proceed to the next step.</li><li>To import the Microsoft project as a subset of an active, existing project or task, click the reference lookup icon (<a href="http://wiki.servicenow.com/index.php?title=File:Icon-referencelookup.png" rel="nofollow" target="_blank"><img alt="Icon-referencelookup.png" class="jiveImage" height="16" src="http://wiki.servicenow.com/images/b/bd/Icon-referencelookup.png" style="border-width: medium; border-style: none;" width="18"/></a>) under <strong>Choose an existing project to import into</strong> and select a project or task. Only active projects appear in the list.</li></ul></li><li>Click <strong>Import</strong>.</li></ol><p style="min-height: 8pt; height: 8pt; padding: 0px;">  </p><p><a _jive_internal="true" href="/servlet/JiveServlet/showImage/38-3249-11822/import_project_post.png"><img  alt="import_project_post.png" class="image-0 jive-image" height="328" src="cd6edc4edb5c9fc03eb27a9e0f961918.iix" style="height: auto; display: block; margin-left: auto; margin-right: auto;" width="341"/></a></p><p style="min-height: 8pt; height: 8pt; padding: 0px;">  </p><p>If you use the Microsoft Project file to create a new project, fields that are mapped between the applications are imported. For example, the information in the Microsoft Project <strong>Task Name</strong> field is added to the ServiceNow Project <strong>Short description</strong> field and the <strong>Actual Start</strong> information is added to the <strong>Actual start date</strong> field.</p><p style="min-height: 8pt; height: 8pt; padding: 0px;">  </p><p>For more details about the information converted during an import, see <span style="text-decoration: underline;"><a class="jive-link-external-small" href="http://wiki.servicenow.com/index.php?title=Importing_and_Exporting_Projects#Importing_Projects" rel="nofollow" target="_blank"><span style="color: #2989c5; text-decoration: underline;">Importing Projects</span></a></span> in the ServiceNow product documentation.</p><p style="min-height: 8pt; height: 8pt; padding: 0px;">  </p><p>Extra bonus - if necessary, you can always <span style="text-decoration: underline;"><a class="jive-link-external-small" href="http://wiki.servicenow.com/index.php?title=Importing_and_Exporting_Projects#Exporting_Projects_to_Microsoft_Project" rel="nofollow" target="_blank"><span style="color: #2989c5; text-decoration: underline;">export a project</span></a></span> from ServiceNow Project back to Microsoft Project.</p>
