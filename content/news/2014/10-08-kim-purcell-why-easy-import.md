<p>If you have<a title="k-external-small" class="jive-link-external-small" href="http://wiki.servicenow.com/index.php?title=Easy_Import" rel="nofollow" target="_blank"> set up your Easy Import </a>to import specific fields from a record, such as work notes and comments, but these are just not showing up, then you may want to tell a certain system property to stop ignoring your conditions.</p><p style="min-height: 8pt; height: 8pt; padding: 0px;">  </p><p>Easy Import is a simplified import process that provides a template that contains only the columns you want to import from a list. If you are missing columns from your template, check the   <span style="font-family: 'courier new', courier;"><strong>glide.import_template.field_types_to_ignore</strong></span> system property. This property ignores specific fields in the import template.</p><p style="min-height: 8pt; height: 8pt; padding: 0px;">  </p><p>To include comments and work notes, or other missing columns in your import, you'll need to remove the field types from that system property.</p><p style="min-height: 8pt; height: 8pt; padding: 0px;">  </p><h2>How to remove the field types from your system property:</h2><ol><li>In the navigation filter, enter <strong>sys_properties.list</strong>.</li><li>Go to <span style="font-family: 'courier new', courier;"><span style="font-family: 'courier new', courier;">glide.import_template.field_types_to_ignore</span>. </span></li><li>In the <strong>Value</strong> field, remove the <strong>journal</strong> and <strong>journal_input</strong> field types (for work notes and comments):</li></ol><p style="min-height: 8pt; height: 8pt; padding: 0px;">  </p><p><a _jive_internal="true" href="/servlet/JiveServlet/showImage/38-3520-14330/Screen Shot 2014-10-07 at 1.18.26 PM.JPG"><img  alt="Screen Shot 2014-10-07 at 1.18.26 PM.JPG" class="image-1 jive-image" height="287" src="ce7108c2db985fc068c1fb651f961967.iix" style="height: 286.548px; width: 564px;" width="564"/></a></p><p>You can apply the same steps above to remove other field types from this system property.</p><p style="min-height: 8pt; height: 8pt; padding: 0px;">  </p><p>For more information on Easy Imports, see:</p><p><a title="" _jive_internal="true" data-containerid="2097" data-containertype="14" data-objectid="162968" data-objecttype="1" href="/community?id=community_question&sys_id=9d920b21dbd8dbc01dcaf3231f96195a">Where can I find and/or enable Easy Import?</a></p><p><a title="" _jive_internal="true" data-containerid="2927" data-containertype="37" data-objectid="3200" data-objecttype="38" href="/community?id=community_blog&sys_id=7d6dea29dbd0dbc01dcaf3231f961958">Import Feature Missing from UI Context Menu</a></p><p><a title="" _jive_internal="true" data-containerid="2098" data-containertype="14" data-objectid="169744" data-objecttype="1" href="/community?id=community_question&sys_id=50d04f65db98dbc01dcaf3231f961960">Easy Import (Import Template) is not allowing to import comments or work notes</a></p>
