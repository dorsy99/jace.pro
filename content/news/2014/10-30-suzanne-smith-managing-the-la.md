<p style="text-align: center;"><em>"If you talk to a man in a language he understands, that goes to his head. If you talk to him in his own language, that goes to his heart."</em></p><p style="text-align: center;">-Nelson Mandela</p><p style="min-height: 8pt; height: 8pt; padding: 0px;">  </p><p>Making your instance, or even parts of your instance, available in the languages that are comfortable for your users is a big win.</p><p style="min-height: 8pt; height: 8pt; padding: 0px;">  </p><p>ServiceNow has several options to set the language for an instance session. It is important to understand the precedence of the different options.</p><p style="min-height: 8pt; height: 8pt; padding: 0px;">  </p><p>The language seen by the user is determined in this order of precedence:</p><ul><li>language selected at login (this option overrides the options below)</li><li>if no language was selected at login, the language selected in the header bar language picker is used</li><li>if no language was selected at login or in the language picker, the user language setting in the User [sys_user] table is used</li><li>if none of the options above are set, the system default language is used</li></ul><p style="min-height: 8pt; height: 8pt; padding: 0px;">  </p><h3>To enable language selection at login:</h3><p style="min-height: 8pt; height: 8pt; padding: 0px;">  </p><ol><li>Navigate to <strong>System Properties &gt; UI Properties</strong>.</li><li>Find the property, <strong>Show the language select box on the login page to allow the user to specify the language they would like to be logged in with</strong>.</li><li>Select <strong>Yes</strong>.<br/>The <strong>Language</strong> option is displayed on the login screen.<br/><a _jive_internal="true" href="/servlet/JiveServlet/showImage/38-3584-15127/internationalization4.png"><img  alt="internationalization4.png" class="image-0 jive-image" height="149" src="1567dccadb945304b322f4621f96198e.iix" style="height: auto;" width="200"/></a></li></ol><h3>To enable the language picker on the header bar, activate one or more Internationalization plugins:</h3><p style="min-height: 8pt; height: 8pt; padding: 0px;">  </p><ol><li>Navigate to <strong>System Definition &gt; Plugins</strong>.</li><li>Scroll through the list to find the plugins beginning with <strong>I18N</strong>.</li><li>Click a plugin name. For example, <strong>I18N: German Translations</strong>.<br/><a _jive_internal="true" href="/servlet/JiveServlet/showImage/38-3584-15119/internationalization2.png"><img  alt="internationalization2.png" class="image-1 jive-image" height="230" src="62665082db989704ed6af3231f961987.iix" style="height: auto;" width="400"/></a></li><li>In <strong>Related Links</strong>, click <strong>Activate/Upgrade</strong>.</li><li>In the Eureka version of ServiceNow, click the gear in the header bar to use the language picker.<br/><a _jive_internal="true" href="/servlet/JiveServlet/showImage/38-3584-15128/internationalization5.png"><img  alt="internationalization5.png" class="image-0 jive-image" height="354" src="0c67b37ddb90d3049c9ffb651f9619a2.iix" style="height: auto;" width="300"/></a><br/><br/><br/></li></ol><h3>To specify a language for a user:</h3><p style="min-height: 8pt; height: 8pt; padding: 0px;">  </p><ol><li>Navigate to <strong>User Administration &gt; Users</strong>.</li><li>In <strong>Language</strong>, select an option. (If the <strong>Language</strong> field is not visible, <a class="jive-link-external-small" href="http://wiki.servicenow.com/index.php?title=Personalizing_Forms" rel="nofollow" target="_blank"><span style="text-decoration: underline;">personalize the form</span></a>.)<br/><a _jive_internal="true" href="/servlet/JiveServlet/showImage/38-3584-15121/internationalization1.png"><img  alt="internationalization1.png" class="image-1 jive-image" height="198" src="c6aab046db1c9344e9737a9e0f9619c2.iix" style="height: auto;" width="300"/></a></li></ol><p style="min-height: 8pt; height: 8pt; padding: 0px;">  </p><h3>To change the system default language:</h3><p style="min-height: 8pt; height: 8pt; padding: 0px;">  </p><ol><li>Navigate to <strong>System Properties &gt; System Localization</strong>.</li><li>Find the property, <strong>Default language for the system (two character values).<br/></strong></li><li>Enter the two character value code for the language you want to set as the default.<br/><a _jive_internal="true" href="/servlet/JiveServlet/showImage/38-3584-15120/internationalization3.png"><img  alt="internationalization3.png" class="image-0 jive-image" height="140" src="3f6d954adbd8130468c1fb651f96193e.iix" style="height: auto;" width="300"/></a></li></ol><p style="min-height: 8pt; height: 8pt; padding: 0px;">  </p><p>Here are some additional tips and tricks when setting instance language options:</p><p style="min-height: 8pt; height: 8pt; padding: 0px;">  </p><ul><li>Right-to-left languages such as Hebrew can be used, but only certain parts of the interface are translated for these languages. For more information, see <a class="jive-link-external-small" href="http://wiki.servicenow.com/index.php?title=Language_Internationalization#Languages" rel="nofollow" target="_blank"><span style="color: #0000ff; text-decoration: underline;">Support for Translating Right-to-Left Languages</span></a>.</li><li>ServiceNow offers twenty I18N internationalization plugins, but if the language you need is not available, you can translate an instance yourself using the information in <a class="jive-link-external-small" href="http://wiki.servicenow.com/index.php?title=Translating_an_Instance" rel="nofollow" target="_blank"><span style="color: #0000ff; text-decoration: underline;">Translating an Instance</span></a>.</li><li>Thinking about <a class="jive-link-external-small" href="/community?id=community_question&sys_id=bffecba1dbdcdbc01dcaf3231f961931" rel="nofollow" target="_blank"><span style="color: #0000ff;">translating service catalog variables</span></a>?
