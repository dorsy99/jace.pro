<p>You can create new modules and <a title="" _jive_internal="true" href="/community/service-automation-platform/user-interface/blog/2015/08/27/save-time-by-setting-up-your-mobile-ui-favorites">set up favorites on the Mobile UI</a> by configuring the URLs that they navigate to. Using specific URLs formats in ServiceNow, you can create modules, favorites, and UI Actions that route to the correct pages.</p><p></p><p>When using the mobile interface to interact with your instance, you will notice a certain URL format which is kept throughout the whole navigation: <strong>type_of_link/table/parameters</strong>. Think of this format as the main ingredient to your configured <a title="" _jive_internal="true" href="/community/service-automation-platform/user-interface/blog/2015/09/02/mobile-url-navigation">navigation URLs </a>you create for your phone or tablet. You can add other "ingredients" to compliment your main ingredient to build your customized navigation.</p><p></p><p>Customized navigation is used when creating:</p><table border="1"><tbody><tr><td><p>Menu Modules: define which modules are available to mobile</p><p>users at the application menu or module level.</p></td><td><img   alt="Screen Shot 2015-09-02 at 3.01.39 PM.png" class="image-0 jive-image" height="277" src="52b7150adb98d344e9737a9e0f96199e.iix" style="height: 277px; width: 176.041px;" width="176"/></td></tr><tr><td><p>Homepage Favorites:   is a list of icons that appears on your mobile homepage.</p><p>These can be used to create a shortcut to any record, list, form or catalog item.</p></td><td><img   alt="Screen Shot 2015-09-02 at 3.01.57 PM.png" class="image-1 jive-image" height="277" src="a9873802dbd097041dcaf3231f9619a0.iix" style="height: 276.935px; width: 176px;" width="176"/></td></tr><tr><td>UI Actions: using the function action.setRedirectURL()</td><td><img   alt="Screen Shot 2015-09-02 at 3.01.12 PM.png" class="jive-image image-2" height="275" src="f216d80adb189fc068c1fb651f96199c.iix" style="height: 274.758px; width: 175px;" width="175"/></td></tr></tbody></table><p></p><p>More information can be found here: <a href="http://wiki.servicenow.com/index.php?title=Configuring_the_Smartphone_Interface" style="font-size: 10pt; line-height: 1.5em;" title="http://wiki.servicenow.com/index.php?title=Configuring_the_Smartphone_Interface">Configuring the Smartphone Interface - ServiceNow Wiki</a></p><p></p><p>You can use the different navigations to create direct links to important aspects of your instance. This allows you to manage the helm while on-the-go, in a meeting, on a call, or while out to lunch from within the Mobile UI:</p><p></p><table border="1"><tbody><tr><th style="text-align: center; background-color: #6690bc; color: #ffffff; padding: 2px;" valign="middle"><strong>Format</strong></th><th style="text-align: center; background-color: #6690bc; color: #ffffff; padding: 2px;" valign="middle"><strong>Description</strong></th><th style="text-align: center; background-color: #6690bc; color: #ffffff; padding: 2px;" valign="middle"><strong>Example</strong></th></tr><tr><td>list/{tableName}/q:{QUERY}</td><td>Directing to a list with the applied query (optional)</td><td><p><strong>list/incident/q:active=true^priority=1^EQ</strong></p><p>List of incidents where active=true and priority=1</p></td></tr><tr><td>streams/{tableName}/q:{QUERY}</td><td>Directing to Journal streams of the table records with the applied query (optional)</td><td><p><strong>streams/incident/q:active=true^priority=1^EQ</strong></p><p>Journal stream of the incident table where active=true and priority=1</p></td></tr><tr><td>form/{tableName}/{sysId}</td><td>Directing to a record form</td><td><p><strong>form/incident/a9a16740c61122760004fe9095b7ddca</strong></p><p>Record form of incident where sys_id=a9a16740c61122760004fe9095b7ddca<strong><br/></strong></p></td></tr><tr><td>form/{tableName}/-1</td><td>Directing to a new record form</td><td><p><strong>form/incident/-1</strong></p><p>Creating a new incident record<strong><br/></strong></p></td></tr><tr><td>catalog/home/{catalogSysId}</td><td>Directing to a specific catalog</td><td><p><strong>home/e0d08b13c3330100c8b837659bba8fb4</strong></p><p>Catalog homepage of the catalog where sys_id=home/e0d08b13c3330100c8b837659bba8fb4 (Service Catalog)</p></td></tr><tr><td><p>catalog/category/{categorySysId}/</p><p>{categoryName}</p></td><td>Directing to a specific catalog category</td><td><p><strong>catalog/category/d258b953c611227a0146101fb1be7c31/</strong></p><p><strong>Hardware</strong></p><p>Catalog category 'Hardware'</p></td></tr><tr><td><p>catalog_item/item/{categorySysId}/</p><p>{categoryName}/<span style="font-size: 10pt; line-height: 1.5em;">{itemSysId}</span></p></td><td>Directs to a specific catalog item</td><td><p><strong>catalog_item/item/d258b953c611227a0146101fb1be7c31/</strong></p><p><strong>Hardware/</strong><strong>0d08837237153000158bbfc8bcbe5d02</strong></p><p>Catalog item form category 'Hardware' where item sys_id=<span style="font-size: 13.3333330154419px;">0d08837237153000158bbfc8bcbe5d02</span></p></td></tr><tr><td>help/{mobileHelpPath}</td><td>Directs to a Mobile Help page</td><td><p><strong>help/defining_tags</strong></p><p>Mobile Help page 'Defining Tags'</p></td></tr><tr><td>view/{UIMacroName}</td><td>Renders a customized UI Macro</td><td><p>view/test_mobile</p><p>Rendering the UI Macro named test_mobile</p></td></tr></tbody></table><p><strong><br/></strong></p><p>When directing to a list or stream (the first two options), you can add a query in order to get filtered records. It is quite tricky to build the query yourself so here is a tip to add the query: <strong>Always use the module form in order to create the query.</strong></p><p></p><h2>Using the Module form to Create a query:</h2><p>1. Go to <strong>System Mobile UI &gt; Navigator Apps. </strong>Enter one of the already existing Applications and create a new module in the <strong>Modules</strong> related list.</p><p><strong><img   alt="sn module query1.jpg" class="image-0 jive-image" src="ea0a518edbd8d344e9737a9e0f96194f.iix" style="height: 360px; width: 620px; display: block; margin-left: auto; margin-right: auto;"/><br/></strong></p><p>2. Select the table in the <strong>Table</strong> field, and <span style="font-size: 13.3333px;">build the required filter </span>using the <strong>Filter Builder</strong>. Then save your changes.</p><p><img   alt="sn module query 2.jpg" class="image-1 jive-image" src="81fefccedb5097049c9ffb651f961900.iix" style="height: 272px; width: 620px; display: block; margin-left: auto; margin-right: auto;"/></p><p>3. In the Mobile user interface, navigate to the temporary module you just created, and copy the URL part that you need form the browser address bar.</p><p><img   alt="Screen Shot 2015-09-02 at 3.01.39 PM.png" class="jive-image image-6" height="250" src="18133f71db98df04e9737a9e0f9619f0.iix" style="height: 250px; width: 158.882px; display: block; margin-left: auto; margin-right: auto;" width="159"/></p><p><img   alt="example url.jpg" class="image-0 jive-image" src="1b091c4edbd45304b322f4621f96197d.iix" style="height: 21px; width: 620px;"/></p><p>4. Now you can use this url also as a favorite link. For example:   <strong>list/incident/q:caller_idDYNAMIC90d1921e5f510100a9ad2572f2b477fe%5Eshort_descriptionLIKETTT%5EEQ</strong></p><p></p><p><strong>A few things to note...</strong></p><ul><li>When having a query in the URL, you have probably notices that it can be dynamic, by containing a javascript function.</li></ul><p>         For example: <span style="line-height: 1.5em; font-size: 8pt;">list/incident/q:sys_created_onONThis%20month@javascript:gs.beginningOfThisMonth()@javascript:gs.endOfThisMonth()%5EEQ</span></p><ul><ul><ul><li>It means, list of incidents which were created this month.</li></ul></ul><li>The url always starts with an exclamation mark '!'. The exclamation mark is being added by the platform, therefore you don't need to add it.<ul><li>For example:<strong><a title="k-external-small" class="jive-link-external-small" href="http://community.servicenow.com/" rel="nofollow" target="_blank">http://</a>&lt;instance_name&gt;.service-now.com/$m.do#/<span style="color: #ff0000;">!</span>list/incident/q:active=true%5EEQ</strong></li></ul></li><li>When navigating by navigation menu modules or favorites, the platform mechanism was designed in a way that you are kept always inside the instance.</li></ul><p><span style="font-size: 10pt; line-height: 1.5em;">               Therefore, you always have to use relative URLs and not absolute ones.</span></p><ul><ul><li>For example: <span style="font-size: 10pt; line-height: 1.5em;">list/incident/q:active=true%5EEQ</span><ul><li>And not:<a title="k-external-small" class="jive-link-external-small" href="http://community.servicenow.com/" rel="nofollow" target="_blank">http://</a>&lt;instance_name&gt;.service-now.com/$m.do#/!list/incident/q:active=true%5EEQ</li></ul></li></ul><li>You can <strong>not</strong> direct navigation menu modules and favorites to external links. UI Actions are a different story, since you are using the function action.setRedirectURL(), where you can send an absolute URL as the function parameter.</li></ul><p></p><p>In this world where we are being stretched thinner and are expected to have a great output, we need to utilize the resources we have. ServiceNow mobile and tablet capabilities are a great way to get more done in the office and still venture away from your desk.</p><p></p><p>Additional resources on using the ServiceNow mobile interface:</p><p><a __default_attr="4495" __jive_macro_name="blogpost" class="jive_macro jive_macro_blogpost" data-orig-content="Debugging the Mobile UI from your Desktop" href="/community?id=community_blog&sys_id=4e0deaa5dbd0dbc01dcaf3231f961914" modifiedtitle="true" title="Debugging the Mobile UI from your Desktop">Debugging the Mobile UI from your Desktop</a></p><p><a __default_attr="4479" __jive_macro_name="blogpost" class="jive_macro jive_macro_blogpost" data-orig-content="List Filtering in the Mobile UI" href="/community?id=community_blog&sys_id=5ebc2e25dbd0dbc01dcaf3231f96191d" modifiedtitle="true" title="List Filtering in the Mobile UI">List Filtering in the Mobile UI</a></p><p><a title="k-external-small" class="jive-link-external-small" href="https://hi.service-now.com/kb_view.do?sysparm_article=KB0551074" rel="nofollow" style="font-weight: inherit; font-style: inherit; font-family: inherit; color: #3778c7; text-decoration: underline;" target="_blank">Mobile View Basics (KB0551074)</a></p>
