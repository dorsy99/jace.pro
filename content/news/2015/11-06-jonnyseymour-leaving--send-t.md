But, wait.</p><p></p><h3>Example #1: I logged in as "aileen.mottern" and create a new incident e.g. INC0010020</h3><p>When the notification fired:</p><ul><li>"aileen.mottern" was NOT in the recipient list.</li><li>"alejandra.prenatt" was is in the recipient list.</li></ul><p></p><p>This is because "aileen.mottern" was the user creating the incident thus the event creator. As the notification 'Send to event creator' is unchecked, it will remove it from the recipient list. From the list of "aileen.mottern" and "alejandra.prenatt", the event creator is removed.</p><p></p><p></p><p>Reviewing on the sysevent table, you can see the User name for the event.</p><p><img   alt="event creator.jpg" class="image-6 jive-image" src="dffc7486db9497041dcaf3231f9619a6.iix" style="width: 620px; height: 199px; display: block; margin-left: auto; margin-right: auto;"/></p><p></p><h3>Example #2: I logged in as "alejandra.prenatt," opened the incident created e.g. INC0010020 and add a comment, then Saved.</h3><p>When the notification fired:</p><ul><li>"aileen.mottern" is in the recipient list.</li><li>"alejandra.prenatt" is NOT in the recipient list.</li></ul><p></p><p>This is because "alejandra.prenatt" was the user making the change thus the event creator. As the notification 'Send to event creator' is unchecked, it will remove it from the recipient list. From the list of "aileen.mottern" and "alejandra.prenatt." the event creator is removed.</p><p></p><p>Reviewing on the sysevent table, you can see the User name for the event.</p><p><img   alt="event creator updated.jpg" class="image-7 jive-image" src="ba2ed046dbdc9704ed6af3231f96194d.iix" style="width: 620px; height: 206px; display: block; margin-left: auto; margin-right: auto;"/></p><h3>Example #3: I logged in as the Administrator, opened the incident created e.g. INC0010020 and add a comment, then Save.</h3><p>When the notification fired:</p><p>"aileen.mottern" and "alejandra.prenatt" are in the recipient.</p><p></p><p>This is because "System Administrator" was the user making the change, thus the event creator.
