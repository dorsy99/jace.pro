<p>This article has an example method to import Incidents without History or relationship information from another ITSM application. There are many different ways to import data in ServiceNow and this is one example method. For your imports you will need to adjust accordingly. <br /><!--break--><br /><br />Note: It should be discouraged to import old Incidents, especially a large number of Incidents that can add a performance burden on a new system. However it sometimes is necessary to import a selected number of incidents. <br /><br />Wiki Articles for reference:<br /><br />http://wiki.servicenow.com/index.php?title=Import_Sets<br
