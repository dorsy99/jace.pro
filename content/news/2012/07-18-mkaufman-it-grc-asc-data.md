Import Set Table:<br /><br /><ul><li>Label: GRC ASC Data Upload</li></ul><br /><br />After Import Set Table is created, Click <strong>Create Transform Map</strong> link.<br /><br /></li><li><strong>Transform Map</strong><br /><br />Setup Transform Map:<br /><ul><li>Name: GRC ASC Data Upload</li><li>Source Table: GRC ASC Data Upload</li><li>Target Table: Authoritative Source Content</li><li>Run Business Rules: false</li><li>Enforce Mandatory fields: No</li><li>Copy empty fields: false</li></ul><br />Transform Map (Field Map)<br /><br /><br /><br /><br /><br /><br /><br
