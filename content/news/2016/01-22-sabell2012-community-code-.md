<p><span style="font-family: arial, helvetica, sans-serif; font-size: 12pt;">Here is a question I get asked a <span style="text-decoration: underline;">LOT</span> in my Discovery and Orchestration classes:   How do I easily get rid of a MID Server that I have (pick one):</span></p><p></p><ul><li><span style="font-family: arial, helvetica, sans-serif; font-size: 12pt;">Accidentally installed!</span></li><li><span style="font-family: arial, helvetica, sans-serif; font-size: 12pt;">Needed to rename!</span></li><li><span style="font-family: arial, helvetica, sans-serif; font-size: 12pt;">Changed a wrapper configuration! (<span style="color: #000000;">If you make changes to the wrapper-override.conf file you will need to remove and re-add the MID Server service to get the changes to take affect.)</span></span></li><li><span style="font-family: arial, helvetica, sans-serif; font-size: 12pt;">Other!</span></li></ul><p></p><p><span style="font-family: arial, helvetica, sans-serif; font-size: 12pt;">So, here is a quick-and-clean way to do the deed:</span></p><p></p><p><span style="color: #000000; font-family: arial, helvetica, sans-serif; line-height: 1.5; font-size: 12pt;">1. From a Microsoft Windows Start</span></p><p></p><p style="padding-left: 30px;"><span style="color: #000000; font-family: arial, helvetica, sans-serif; line-height: 1.5; font-size: 12pt;">a. Type <strong>cmd</strong>.   You will see the cmd.exe appear in the search list.</span></p><p></p><p><span style="color: #000000; font-family: arial, helvetica, sans-serif; line-height: 1.5; font-size: 12pt;"><img   class="image-2 jive-image" src="0cd844c2db981344e9737a9e0f961986.iix" style="width: auto; height: auto; display: block; margin-left: auto; margin-right: auto;"/></span></p><p style="padding-left: 30px;"></p><p style="padding-left: 30px;"><span style="color: #000000; font-family: arial, helvetica, sans-serif; font-size: 12pt;">b. Right-click on cmd.exe and choose "Run as administrator".   This will keep a bunch of annoying popup boxes from appearing.</span></p><p style="padding-left: 30px;"></p><p><span style="color: #000000; font-family: arial, helvetica, sans-serif; font-size: 12pt;"><img   class="image-1 jive-image" src="1f440582dbd013043eb27a9e0f9619ac.iix" style="width: auto; height: auto; display: block; margin-left: auto; margin-right: auto;"/></span></p><p></p><p style="padding-left: 30px;"><span style="color: #000000; font-family: arial, helvetica, sans-serif; font-size: 12pt;">c. The command window will be displayed.</span></p><p></p><p><span style="color: #000000; font-family: arial, helvetica, sans-serif; font-size: 12pt;"><img   class="image-3 jive-image" src="bba5e375dbd41704ed6af3231f9619e1.iix" style="width: 620px; height: 313px; display: block; margin-left: auto; margin-right: auto;"/></span></p><p></p><p style="padding-left: 30px;"></p><p><span style="color: #000000; font-family: arial, helvetica, sans-serif; font-size: 12pt;">2. Change the directory to your MID Server location.   Mine is midserver1, and you want to be in the agent directory to issue the commands.</span></p><p></p><p style="padding-left: 30px;"><span style="color: #000000; font-family: arial, helvetica, sans-serif; font-size: 12pt;">a. Type: <strong>cd \midserver1\agent</strong>         - and press enter</span></p><p style="padding-left: 30px;"><span style="color: #000000; font-family: arial, helvetica, sans-serif; font-size: 12pt;">b. Type: <strong>dir               </strong>- to bring up a directory</span></p><p></p><p><span style="color: #000000; font-family: arial, helvetica, sans-serif; font-size: 12pt;"><img   class="jive-image image-4" src="bae7414adb50d304b322f4621f9619b2.iix" style="width: 620px; height: 434px; display: block; margin-left: auto; margin-right: auto;"/></span></p><p></p><p><span style="color: #000000; font-family: arial, helvetica, sans-serif; font-size: 12pt;">3. Now issue the stop command to kill the MID Server process.</span></p><p></p><p style="padding-left: 30px;"><span style="color: #000000; font-family: arial, helvetica, sans-serif; font-size: 12pt;">a. <strong>stop.bat</strong></span></p><p></p><p><span style="color: #000000; font-family: arial, helvetica, sans-serif; font-size: 12pt;"><img   class="image-5 jive-image" src="2ea7ec4adb9457049c9ffb651f96199f.iix" style="width: 620px; height: 247px; display: block; margin-left: auto; margin-right: auto;"/></span></p><p></p><p></p><p><span style="color: #000000; font-family: arial, helvetica, sans-serif; font-size: 12pt;">4. To delete the MID Server you will need to issue the sc delete command.   The midserver name will be out of the box: snc_mid</span></p><p></p><p style="padding-left: 30px;"><span style="color: #000000; font-family: arial, helvetica, sans-serif; font-size: 12pt;">a. <strong>sc delete snc_mid</strong></span></p><p></p><p><span style="color: #000000; font-family: arial, helvetica, sans-serif; font-size: 12pt;"><img   class="image-6 jive-image" src="2216c8cadb1c5fc03eb27a9e0f96191d.iix" style="width: 620px; height: 247px; display: block; margin-left: auto; margin-right: auto;"/></span></p><p></p><p style="padding-left: 30px;"><span style="color: #000000; font-family: arial, helvetica, sans-serif; font-size: 12pt;"><strong>Note</strong>: snc_mid is the actual name of the MID Server service as indicated in the wrapper-override.conf file.</span></p><p style="padding-left: 30px;"></p><p style="padding-left: 30px;"><span style="color: #000000; font-family: arial, helvetica, sans-serif; font-size: 12pt;"><strong>Note</strong>:   This removes the MID Server service and now you are ready to start fresh.</span></p><p></p><p><span style="font-family: arial, helvetica, sans-serif; font-size: 12pt;">5. Now you can make modifications to the configuration or whatever.   Running start.bat will re-install the MID Server service with your changes.</span></p><p></p><p style="padding-left: 30px;"><span style="color: #000000; font-family: arial, helvetica, sans-serif; line-height: 1.5; font-size: 12pt;">a. <strong>start.bat</strong></span></p><p style="padding-left: 30px;"></p><p><span style="font-family: arial, helvetica, sans-serif; font-size: 12pt;"><img   class="image-7 jive-image" src="d4d66779db5493049c9ffb651f9619e5.iix" style="width: 620px; height: 247px; display: block; margin-left: auto; margin-right: auto;"/></span></p><p></p><p style="padding-left: 30px;"><span style="color: #000000; font-family: arial, helvetica, sans-serif; font-size: 12pt;"><strong>Note</strong>: The start.bat will re-install the MID Server service.</span></p><p></p><p></p><p><span style="color: #000000; font-family: arial, helvetica, sans-serif; font-size: 12pt;">And you are done!
