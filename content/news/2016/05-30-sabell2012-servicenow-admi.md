<p style="font-family: arial, sans-serif; color: #666666;"><span style="font-weight: inherit; font-family: inherit; font-style: inherit;"><strong>NOTE</strong></span>: ON APRIL 1, 2016 ACCENTURE COMPLETED THE ACQUISITION PROCESS OF CLOUDSHERPAS.   AT THAT TIME THE CLOUDSHERPAS BLOG SITES WERE DOWNED FOREVER.</p><p></p><p style="font-family: arial, sans-serif; color: #666666;">THIS IS THE RE-PUBLICATION OF MY ARTICLE FROM <span style="font-weight: inherit; font-family: inherit; font-style: inherit;"><strong>June 10, 2014</strong></span> ON THE CLOUDSHERPAS SERVICENOW SCRIPTING 101 BLOG.</p><p style="font-family: arial, sans-serif; color: #666666;">____________________________________________________________________________</p><p></p><p><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif; color: #000000;">One of our implementation specialists recently posed an interesting question: is it possible to create a link that jumps to the top of the page on a Service Catalog Item in ServiceNow? </span></p><p></p><p><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif; color: #000000;">He was working on several long Catalog Item forms that required users to scroll all the way back to the top once they reached the bottom of the page. Recognizing that getting back to the top could be painful if there were a significant number of text field variables embedded in the form, he thought why not add a link at the bottom that goes directly to the top of the page?</span></p><p></p><p><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif; color: #000000;">My first thought was, you can't do that without modifying the out-of-the-box code. However, this is not a great option because when it comes to ServiceNow, if you touch it, you own it. In other words, ServiceNow will skip any modified out-of-the-box code when upgrading to a new release.</span></p><p></p><p><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif; color: #000000;">So the solution had to be something that would work inside of the existing framework for Service Catalog. And then it struck me! Why not create a UI Macro, a Service Catalog Macro to use it. and then add that to a Catalog Item? It couldn't be that simple… could it? It was, and the solution worked great!</span></p><p></p><p><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif; color: #000000;">To create your own "Top of Page" link for a Service Catalog page, just follow these steps:</span></p><p></p><p></p><p><span style="color: #000000; font-size: 12pt; font-family: arial, helvetica, sans-serif;"><strong>1. Create a new UI Macro</strong></span></p><p></p><p style="padding-left: 30px;"><span style="color: #000000; font-size: 12pt; font-family: arial, helvetica, sans-serif;">a. In the navigation filter type: UI Macros</span></p><p style="padding-left: 30px;"></p><p style="padding-left: 30px;"><span style="color: #000000; font-size: 12pt; font-family: arial, helvetica, sans-serif;">b. Click on the "UI Macros" link.</span></p><p></p><p style="padding-left: 30px;"><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif;"><span style="color: #000000;">   </span><span style="color: #000000;"><img class="jive-image" height="190" src="https://lh5.googleusercontent.com/_s-yBi29Py713JX3pj1pBArGc7TeT1IMsyXXQzFL3s7fdErkaE794C8THNRiMLfrVEjnSAWluBAksL4f3JtDyv0bJYlj25OoDtrCeSzGITdbDRoX0YXc4UMOv5hqisjM4hwJW8Ao7vmiz1hL" style="border-style: none;" width="233"/></span></span></p><p></p><p style="padding-left: 30px;"><span style="color: #000000; font-size: 12pt; font-family: arial, helvetica, sans-serif;">c. The UI Macro list view will appear.</span></p><p style="padding-left: 30px;"></p><p style="padding-left: 30px;"><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif; color: #000000;">d. Click "New" to create the new UI Macro. The new UI Macro form will appear with the XML partly filled in.</span></p><p><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif; color: #000000;">   </span></p><p><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif; color: #000000;"><img class="jive-image" height="385" src="https://lh5.googleusercontent.com/IHmqhkwUY9Wbm03effpsMI4081E5TeIa2u0r1HeEJpnZ1fholO5fPXNIRb2TJlb-xF3j0KWyS2ahkKZmSNNcVHUdvOjGwHkUB6VehV4AEHG3LgYbBvvkZpFwiprDAdUx6rHnhOIv7A_oDjht" style="border-style: none; display: block; margin-left: auto; margin-right: auto;" width="665"/></span></p><p style="padding-left: 30px;"></p><p style="padding-left: 30px;"><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif; color: #000000;">e. In the "Name" field, type something like "sc_top_of_page_link" for easy identification.</span></p><p style="padding-left: 30px;"></p><p style="padding-left: 30px;"><span style="color: #000000; font-size: 12pt; font-family: arial, helvetica, sans-serif;">f. Add the following code in the XML field between the two jelly tags:</span></p><p style="padding-left: 30px;"></p><p style="padding-left: 60px;"><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif;"><span style="color: #0000ff;">&lt;a href="#top"&gt;Goto top of page&lt;/a&gt;</span><span style="color: #000000;"><br class="kix-line-break"/><br class="kix-line-break"/></span></span></p><p style="padding-left: 60px;"><span style="color: #000000; font-size: 12pt; font-family: arial, helvetica, sans-serif;">The XML code should look like this when you are done:</span></p><p style="padding-left: 30px;"></p><p style="padding-left: 60px;"><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif;"><span style="color: #0000ff;">&lt;?xml version="1.0" encoding="utf-8" ?&gt;</span><span style="color: #000000;"><br class="kix-line-break"/></span><span style="color: #0000ff;">&lt;j:jelly trim="false" xmlns:j="jelly:core" xmlns:g="glide" xmlns:j2="null" xmlns:g2="null"&gt;</span><span style="color: #000000;"><br class="kix-line-break"/></span><span style="color: #0000ff;">&lt;a href="#top"&gt;Goto top of page&lt;/a&gt;</span><span style="color: #000000;"><br class="kix-line-break"/></span><span style="color: #0000ff;">&lt;/j:jelly&gt;</span></span></p><p style="padding-left: 30px;"></p><p style="padding-left: 60px;"><span style="color: #000000; font-size: 12pt; font-family: arial, helvetica, sans-serif;"><strong>NOTE</strong>: You can also create a similar #bottom or #middle UI Macro.</span></p><p style="padding-left: 30px;"></p><p style="padding-left: 30px;"></p><p style="padding-left: 30px;"><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif; color: #000000;">g. You're now done with the UI Macro! It should look something like this:</span></p><p></p><p><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif; color: #000000;"><img class="jive-image" height="266" src="https://lh5.googleusercontent.com/xH3egAj_ihE7oAAG6r8Q4-jEJIgHsfLmo-qVO9y5URN2E5zmjrOT2PUODBxZDK3nVqOKRNn_fytweweA29W8aEt8eZskrNwzJyGNnpv-AY_GuvajkiBYTKpz_wP2v7Zy7qYkwwfEEjAZzhzb" style="border-style: none; display: block; margin-left: auto; margin-right: auto;" width="665"/></span></p><p></p><p style="padding-left: 30px;"><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif; color: #000000;">h. Click the Submit button. This will take you back to the UI Macro list view.</span></p><p style="padding-left: 30px;"></p><p style="padding-left: 30px;"><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif; color: #000000;">i. In the list view, double click and set the Category to Service Catalog. This identifies it as a Service Catalog UI Macro.</span></p><p style="padding-left: 30px;"></p><p><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif; color: #000000;"><img class="jive-image" height="67" src="https://lh5.googleusercontent.com/nwVF3L176KYY5VV88w-BH7QlC7KztRNfrUMRMeGARJXnXn1EEMnhbc2tyE1OA-NCGaXxYZGYvJI4FzvEro1G_8kvEAxCQJNtJGeOVup3RBq7y2tf4d-bc4Bm_0m8IZSBM_4gt_k3y23zG_SX" style="border-style: none; display: block; margin-left: auto; margin-right: auto;" width="665"/></span></p><p><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif; color: #000000;"><br class="kix-line-break"/><br class="kix-line-break"/></span></p><p><span style="color: #000000; font-size: 12pt; font-family: arial, helvetica, sans-serif;"><strong>2. Create a new Service Catalog Variable</strong></span></p><p></p><p style="padding-left: 30px;"><span style="color: #000000; font-size: 12pt; font-family: arial, helvetica, sans-serif;">a. Go to the Navigation filter and type "Catalog Variables"</span></p><p></p><p><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif;"><span style="color: #000000;">   </span><span style="color: #000000;"><img class="jive-image" height="191" src="https://lh5.googleusercontent.com/GT9IxgqyHocXR4kSTUMSUMpso7wou0NglBjPJtq4cqp74aM2Nuo8ELS3Y68icf4At3wsipqhtT80itJKL-h-aBFZcKq4TrqvO6Sh3_Y1gIVwmO_PAHEO9D6VbiNkH1VFvSFdH1EuuCWxvvbz" style="border-style: none; display: block; margin-left: auto; margin-right: auto;" width="555"/></span><span style="color: #000000;">   </span><span style="color: #000000;">   </span></span></p><p></p><p style="padding-left: 30px;"><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif; color: #000000;">b. Click on the "All Variables" link. This will display the Service Catalog Variables list view.</span></p><p style="padding-left: 30px;"></p><p style="padding-left: 30px;"><span style="color: #000000; font-size: 12pt; font-family: arial, helvetica, sans-serif;">c. Click the "New" button to display the new Variable form.</span></p><p></p><p><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif; color: #000000;"><img class="jive-image" height="296" src="https://lh3.googleusercontent.com/5qISyZ1WBAG0t3dTCETCB20u7YVluDiPnCi-QpmbRR8-WLiRyFBaHjApMWBR5AQJXMuG6Q5YVIjgtf0yfxqQ4mfhYK3vInMYZHS83lyragCp-zWtb_uTE-gPreydahU2cle3Ga-pbSLuyuOg" style="border-style: none; display: block; margin-left: auto; margin-right: auto;" width="665"/></span></p><p></p><p style="padding-left: 30px;"><span style="color: #000000; font-size: 12pt; font-family: arial, helvetica, sans-serif;">d. Fill in the following fields as designated below, leaving the rest as default.</span></p><p style="padding-left: 30px;"></p><p style="padding-left: 60px;"><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif;"><span style="color: #0000ff;"><strong>Type</strong>: Macro</span><span style="color: #0000ff;"><br class="kix-line-break"/></span><span style="color: #0000ff;"><strong>Name</strong>: sr_goto_top_of_page</span><span style="color: #0000ff;"><br class="kix-line-break"/></span><span style="color: #0000ff;"><strong>Order</strong>: 1999</span></span></p><p style="padding-left: 60px;"><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif; color: #0000ff;"><strong>Catalog Item</strong>: &lt;your really long catalog item&gt;</span></p><p style="padding-left: 60px;"><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif;"><span style="color: #0000ff;"><strong>Macro</strong>: &lt;will appear when Type: Macro is chosen&gt;</span><span style="color: #0000ff;"><br class="kix-line-break"/><br class="kix-line-break"/></span></span></p><p style="padding-left: 60px;"><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif; color: #000000;">When you pick the Macro type, the "Macro" field should appear. Click the magnifying glass to bring up a list of Macros and pick the "sc_top_of_page_link" Macro that you created in the first step.</span></p><p style="padding-left: 60px;"></p><p style="padding-left: 60px;"></p><p style="padding-left: 60px;"><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif; color: #000000;">By the way, you are not limited to how many of these you can create. So, if you have a really long Service Catalog page, you can create a couple of these and put one in the middle of the page. Location of the variable is determined by the "Variable Order" field. For the purposes of this article, I will show just the one variable at the end of the Service Catalog page. </span></p><p style="padding-left: 60px;"></p><p style="padding-left: 60px;"></p><p style="padding-left: 60px;"><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif; color: #000000;">Since you want the link to be at the end of the page, set the order to be greater than all the other variables. 1,999 should do the trick nicely!</span></p><p style="padding-left: 60px;"></p><p style="padding-left: 60px;"><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif; color: #000000;">For Catalog Item, choose "Schedule Relocation" because it is a long one and will demonstrate the link nicely. Add a prefix of "sr" to the "Name" field to identify it as a Schedule Relocation variable.</span></p><p></p><p><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif; color: #000000;"><img class="jive-image" height="406" src="https://lh4.googleusercontent.com/I0ZxsJtQAcdlUfXGs3cnuBFNTfmmMR6py6JFoYiJ-ddfdiRIpqZ39fj4jR_1nj9OG_2d6JrN6GtXBdkvFul-KTvourLOVs4QyRPkqU1g9lUAnYBaQR3fnflfOSK2sdt6-2BzaiHWDWvTTUnF" style="border-style: none; display: block; margin-left: auto; margin-right: auto;" width="618"/></span></p><p></p><p style="padding-left: 30px;"><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif; color: #000000;">e. The completed form should look something like this when you are done:</span></p><p></p><p><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif; color: #000000;"><img class="jive-image" height="326" src="https://lh6.googleusercontent.com/IfrObcMJ79qqYL36MXJoJXxYdReMmAByZke13t647PsExhhB5_resVldQl82qnBPJKNr4hTEZkUY-B0EAQo9O-uZAF8VGJaCSYNuO51pga3pqS-YN1iIx0jhvr9l5yvZgYDvrVl80NVHjnDr" style="border-style: none; display: block; margin-left: auto; margin-right: auto;" width="617"/></span></p><p></p><p style="padding-left: 30px;"><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif; color: #000000;">f. Click on the "Submit" button to display the "All Variables" list view.</span></p><p><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif;"><br/><br/></span></p><p><span style="color: #000000; font-size: 12pt; font-family: arial, helvetica, sans-serif;"><strong>3. Test your Catalog Item</strong></span></p><p><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif; color: #000000;">   </span></p><p style="padding-left: 30px;"><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif; color: #000000;">Okay, we're done! Choose "Service Catalog" in the self-service application.</span></p><p style="padding-left: 30px;"></p><p style="padding-left: 30px;"><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif; color: #000000;">c. Click on "Can We Help You?"</span></p><p></p><p><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif; color: #000000;"><img class="jive-image" height="304" src="https://lh4.googleusercontent.com/a-ukr01btw7T9m3LWNitvMmsB-i0uKqpLzykAKIsf2ZXBsavFhuUqzA67H1OfN9xaf-UehI0MPIw7Il8PLzE5qqsn9GxQSyPWWJCwkWDu-DS0G-DIiH4i0h85RAQLa3Kg5as8vjQ1GYQZmFg" style="border-style: none; display: block; margin-left: auto; margin-right: auto;" width="665"/></span></p><p></p><p style="padding-left: 30px;"><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif; color: #000000;">d. Scroll down and click on "Schedule Rotation"</span></p><p></p><p><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif; color: #000000;"><img class="jive-image" height="299" src="https://lh5.googleusercontent.com/-CzzRHquMSZyblY0pcbtrxwShS2KuTq0iW3Nw3gfIuTsoRZHQxaRGnkeV3JU6NweYodNbKAxga5J9QXYmtdFS2vJTMbn7Ma8xuhT76MLMVr2a93Oh_8vFNJwBuXf7pTBV9t-yDkuj7InCbu8" style="border-style: none; display: block; margin-left: auto; margin-right: auto;" width="665"/></span></p><p></p><p style="padding-left: 30px;"><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif; color: #000000;">e.
